# 知识库检索地图

## 使用原则

- 检索是按需取证，不是默认入口。先用当前上下文和可靠原理形成初步判断；问题本身已经能讲清时直接回答。
- 先完成初步审题，再检索。不要让相似关键词把假问题带进旧答案。
- 当前只有文字症状或第三方转述、因果尚未通过控制变量定位时，不检索具体插件处方来评选原因；检索只能用于帮助设计验证。取得实际结果后，再查对应方法和条件。
- 每次优先读取最相关的 1-3 个文件，必须读正文和上下文。
- 案例参数只代表当时素材，不自动推广为固定标准。
- 可以用可靠通用知识或宿主允许的外部资料补足未覆盖的概念、图示、软件事实和 DAW 操作，但不得虚构瑞哥的经验或偏好。
- 不输出身份信息、联系方式、文件路径或无关原始对话。
- 不越过音频证据：可以说明如何验证，不得声称已经听出用户作品的问题。

## 证据与来源仲裁

不要按“本地知识库、网络、模型已有知识”给来源预设固定高低。先检查内容是否对应当前人物、对象、版本、制作阶段和目标，再按下面的证据层级使用：

1. 当前对话中已经明确的事实，以及用户实际操作或听看后的结果。
2. 截图、读数、文件信息、软件文档或其他可以核验的材料。
3. 有明确适用条件的稳定原理和方法。
4. 单个案例、常见做法、参数范围和个人经验。

低层级内容可以帮助提出假设或实验，不能覆盖高层级的当前事实。来源更具体、篇幅更长或关键词更相似，都不代表更适合当前问题。

多个来源冲突时，不投票、不平均参数，也不优先相信本地文件。先找出它们在对象、目标、信号位置、制作阶段、版本和前提上的差异；条件仍不清楚时，给能区分分支的验证动作，或明确说明当前无法确定。

## 来源与表达

- 完整使用可靠方法，默认只输出方法、条件和验证。
- 不展示参考文件名、目录、检索词或读取过程。
- 不把 AI 整理、推断或补充的内容归为瑞哥本人观点。
- 只有当前对话提供明确公开来源，或瑞哥在当前对话确认时，才说明具体归属。

## 知识库位置

知识库位于 Skill 内的 `knowledge/`。只检索当前 Skill 实际包含的文件。

先按问题类型读取最相关的 1-2 份汇总知识：

- `knowledge/公开基础知识.md`：问题校准、回答顺序和通用边界
- `knowledge/公开知识-编曲审美与执行.md`：审美拆解、配器职能、情绪推进和编曲说明
- `knowledge/公开知识-混音概念与信号流.md`：EQ、压缩、空间、插入、发送、总线和检查方法
- `knowledge/公开知识-录音修音与导出.md`：输入监听、Buffer、路由排查、修音和导出

其他补充知识按目录和文件的实际内容检索。

### 编曲审美与音乐语言转译

优先搜索参考方向、配器职能、结构、律动、动态和 AI 生成目标。没有覆盖时，用可靠通用知识继续解释。

### 混音概念与操作答疑

优先搜索概念、操作对象、信号流和真实目标，不只搜索“闷、糊、刺”等症状词。涉及具体工具或瑞哥是否使用过某工具时，没有可靠命中就说明这是通用建议。

### 音乐职业、变现与事业发展

先读取 `references/career-guidance.md`，按目标、已确认事实、待验证判断、缺失条件和下一步证据推进。

当前公开包只使用经过确认和脱敏的职业方法，不直接收录原始咨询对话，也不把 AI 摘要、OpenCode 提炼或外部框架压缩当作瑞哥原话。职业来源优先级为：原始咨询事实与当前对话确认的稳定判断，其次是方法文档；AI 或外部整理只能作为检索线索，不能单独支撑重要结论。

用户的“音乐职业成长档案”属于其私有运行数据，不是知识库。不得检索、复制或发布到 Skill 包；只有用户明确同意并指定目录后，才能按 `references/career-guidance.md` 读写。

## 搜索方法

优先组合对象与目标两个维度，例如：

```bash
rg -l -i --glob '*.md' '人声.*发送|发送.*人声' knowledge
rg -l -i --glob '*.md' '母带.*必须|必须.*母带' knowledge
rg -l -i --glob '*.md' 'Suno|AI.*编曲' knowledge
```

也可运行：

```bash
scripts/search-knowledge.sh '母带|总线压缩' 20
```

## 检索后判断

1. 文件回答的是同一个真问题，还是只有表面关键词相似？
2. 结论是否依赖听审、具体素材或特定软件？
3. 命中内容属于哪一级？
   - **稳定原理**：可以迁移，但仍说明适用条件。
   - **条件方法**：只在相同目标、对象和制作阶段下使用。
   - **案例参数**：只能作为实验起点，不能写成标准答案。
   - **个人经验或推荐**：只有明确公开来源或当前对话确认时才说明归属。
4. 当前回答是否越过听审边界？

检索命中不等于瑞哥本人推荐。没有可靠命中时，使用可靠通用知识或外部资料，不拼凑不相关案例。无论答案来自哪里，都必须重新经过当前对象、证据边界和适用条件检查。
