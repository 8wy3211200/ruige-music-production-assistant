# 开放排列

**核心概念**：将和弦的音符分散在多个八度内，产生宽广的空间感

**具体操作**：
1. 确定和弦的根音位置（通常在较低音区）
2. 将三音、五音、七音等分散到较高音区
3. 在音符之间留出足够的空间
4. 可以跨越多个八度排列音符

**音符示例**：
- C大三和弦开放排列：C3（根音）- G3（五音）- C4（八度）- E4（三音）
- Am7和弦开放排列：A2（根音）- E3（五音）- A3（八度）- C4（三音）- G4（七音）

**适用场景**：
- 需要空间感的编曲
- 电影配乐或氛围音乐
- 新世纪音乐或冥想音乐
- 歌曲的前奏或间奏

**实战案例**：在电影配乐的氛围部分，使用钢琴演奏开放排列的和弦，创造宽广的空间感
