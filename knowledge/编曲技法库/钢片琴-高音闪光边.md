# 钢片琴高音闪光边

**核心概念**：钢片琴适合补“亮但不刺”的高音闪光边。它不是只能做可爱音色，也不是钟琴的简单替代。

**具体操作**：
1. 先判断段落是不是缺高音闪点，而不是缺主体。
2. 只在关键旋律音、和弦转折或段落入口加入钢片琴。
3. 避免长时间密集演奏，保留“闪一下”的辨识度。
4. 可以和竖琴、长笛、弱奏弦乐叠加，形成透明上沿。

**配器示例**：
- 副歌第一句尾音，用钢片琴高八度点一下旋律终止音。
- 梦幻段落里，竖琴分解和弦 + 钢片琴点出和弦高音。

**适用场景**：
- 高音区缺一点亮边
- 梦幻、童话、冰晶感
- 弱奏段落需要精致穿透

**常见误区**：
- 让钢片琴扛大体量中层。
- 从头到尾都在闪，导致廉价和疲劳。
- 把它当成“可爱音色”而不是高音点彩工具。
