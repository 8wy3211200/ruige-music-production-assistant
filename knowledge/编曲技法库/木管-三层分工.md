# 木管三层分工

**核心概念**：木管不要一坨一起上。更实用的理解是上层发光、中层说话和粘合、下层补木质低音。

**具体操作**：
1. 先判断木管这次要做什么：旋律、提亮、填充、过渡，还是低音支撑。
2. 上层用长笛 / 短笛补亮度和空气感。
3. 中层用双簧管、英国管、单簧管承担旋律、对答或粘合。
4. 下层用大管 / 低音大管补木管底盘，不要让低层过碎。

**配器示例**：
- 弦乐铺身体，长笛高八度提亮旋律，单簧管补中间层，大管轻轻托低音。
- 旋律要有说话感时，用双簧管或单簧管，不要整组齐奏。

**适用场景**：
- 木管一加就乱
- 编曲中层空
- 弦乐和铜管之间缺连接

**常见误区**：
- 所有木管都一起干同一件事。
- 以为木管越多越高级。
- 在厚弦乐同音区硬叠木管，导致木管被盖掉。
