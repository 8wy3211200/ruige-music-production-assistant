# Filter滤波器（合成器）

**核心概念**：滤波器控制合成器音色的频率内容，塑造音色特征

**具体操作**：
1. **滤波器类型**：
   - 低通滤波器（LPF）：允许低频通过，衰减高频，温暖柔和
   - 高通滤波器（HPF）：允许高频通过，衰减低频，明亮清晰
   - 带通滤波器（BPF）：允许特定频段通过，衰减两侧，电话音色
   - 陷波滤波器（Notch）：衰减特定频段，允许两侧通过，特殊效果

2. **滤波器参数**：
   - 截止频率（Cutoff）：决定通过/衰减的分界点
   - 谐振（Resonance）：在截止频率处增强，创造峰值或自振荡
   - 斜率（Slope）：衰减陡度，12dB/octave或24dB/octave
   - 驱动（Drive）：输入增益，增加谐波和饱和度

3. **滤波器调制**：
   - 包络调制：随时间变化的滤波器扫描
   - LFO调制：周期性滤波器变化
   - 键盘跟踪：截止频率随音高变化
   - 力度调制：力度影响滤波器开合

**音符示例**：
- LPF扫频pad：C3全音符，包络调制cutoff从200Hz到2000Hz
- HPF pluck：C4八分音符，快速attack包络调制HPF
- BPF lead：C5四分音符，LFO调制cutoff创造哇音效果
- 高谐振bass：C2四分音符，resonance 70%，创造尖锐音头

**适用场景**：
- 塑造音色明亮度
- 创造动态变化
- 模拟真实乐器特性
- 增加音色表现力

**实战案例**：House音乐pluck音色：使用LPF，cutoff 1500Hz，resonance 30%，包络快速调制cutoff从1500Hz到500Hz，创造典型的pluck衰减
