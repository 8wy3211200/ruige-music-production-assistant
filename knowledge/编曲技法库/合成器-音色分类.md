# 音色分类（合成器）

**核心概念**：了解合成器音色的基本分类和应用场景

**具体操作**：
1. **Pad音色**：持续的背景铺垫音色
   - 特点：长衰减、氛围感强、通常使用低通滤波器
   - 应用：背景氛围、弦乐替代、空间填充
   - 参数：长attack、长release、多振荡器、合唱效果

2. **Lead音色**：主旋律演奏音色
   - 特点：明亮、突出、有表现力
   - 应用：主旋律、hook部分、solo
   - 参数：快速attack、中等release、滤波器调制、滑音

3. **Bass音色**：低音部分音色
   - 特点：低频饱满、有冲击力
   - 应用：低音线条、节奏基础
   - 参数：锯齿波或方波、低通滤波器、包络调制

4. **Pluck音色**：弹拨类音色
   - 特点：短促、有音头、节奏感强
   - 应用：节奏伴奏、琶音、装饰音
   - 参数：快速attack、快速decay、零sustain

5. **Keys音色**：键盘类音色
   - 特点：类似钢琴、电钢琴
   - 应用：和声伴奏、旋律演奏
   - 参数：多振荡器、滤波器共鸣、包络控制

6. **FX音色**：效果音色
   - 特点：特殊效果、氛围感
   - 应用：过渡效果、氛围营造
   - 参数：大量效果器、滤波器扫描、LFO调制

**音符示例**：
- Pad：C3全音符，缓慢起音，长衰减
- Lead：C4八分音符旋律线，快速起音
- Bass：C2四分音符，与底鼓对齐
- Pluck：C4十六分音符琶音
- Keys：C3-E3-G3柱式和弦
- FX：C3长音符，滤波器从低到高扫描

**适用场景**：
- 编曲中的不同层次构建
- 根据音乐风格选择合适音色
- 音色搭配和频率分配

**实战案例**：流行歌曲编曲：Pad作为背景层，Keys作为和声层，Pluck作为节奏层，Lead作为旋律层，Bass作为低音层
