# 前中后景分层

**核心概念**：同一时间里，把音乐分成前景、中景、背景。前景负责主旋律或主要乐思，中景负责副旋律、内声部和对答，背景负责和声、伴奏和持续环境。

**具体操作**：
1. 先标出当前 4-8 小节里必须被听见的主角。
2. 把其余乐器分成“托主角”和“铺环境”两类。
3. 前景不要和背景使用同样的音区、节奏密度和音色方向。
4. 如果听起来很满但不清楚，先删背景，不要先加乐器。

**音符 / 配器示例**：
- 前景：小提琴一部 / 单簧管 / 小号负责旋律。
- 中景：中提琴、圆号或木管短句做对答。
- 背景：弦乐长音、Pad 或低音持续支撑。

**适用场景**：
- 用户说“编曲很满但听不清”
- 用户说“配器很厚但不成立”
- 大编制 / 管弦 / 影视配器整体判断

**常见误区**：
- 每一层都写得像主角。
- 只通过音量大小区分主次。
- 背景伴奏和主旋律节奏太像，导致主旋律被吃掉。
