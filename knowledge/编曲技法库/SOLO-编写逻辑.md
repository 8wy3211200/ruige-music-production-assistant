# SOLO编写逻辑

**核心概念**：SOLO需要有清晰的结构和逻辑，不能只是音符的堆砌

**具体操作**：
1. **SOLO结构设计**：
   - **起**（1-2小节）：引入主题，建立调性
   - **承**（3-4小节）：发展主题，增加变化
   - **转**（5-6小节）：高潮部分，技巧展示
   - **合**（7-8小节）：回归主题，圆满结束

2. **音符密度控制**：
   - 开始部分：稀疏，留出空间
   - 发展部分：逐渐增加密度
   - 高潮部分：密集，快速音符
   - 结束部分：稀疏，长音符

3. **音区运用**：
   - 低音区：沉稳，建立基础
   - 中音区：旋律性，歌唱性
   - 高音区：紧张，高潮部分
   - 音区爬升：从低到高创造紧张感

4. **和声跟随**：
   - 强调和弦音（根音、三音、五音、七音）
   - 使用经过音连接不同和弦音
   - 在和弦变化处做相应调整
   - 使用目标音导向下一和弦

**音符示例**：
- 起：C大调，中音区，四分音符和八分音符，强调C、E、G
- 承：增加十六分音符，加入经过音，音区略微上升
- 转：高音区，快速三连音，推弦技巧，强调紧张感
- 合：回到中音区，长音符，解决到主音C

**适用场景**：
- 所有类型的SOLO编写
- 避免SOLO杂乱无章
- 创造有说服力的独奏段落
- 让SOLO服务歌曲整体

**实战案例**：8小节吉他SOLO：1-2小节中音区旋律引入，3-4小节音区上升加入装饰音，5-6小节高音区快速乐句高潮，7-8小节下降解决到主音，最后长延音
