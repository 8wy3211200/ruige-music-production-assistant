# 和弦套路困境

**用户问题**：为什么我的和弦进行听起来总是很套路，缺乏新意？

**问题分析**：
1. 过度依赖常见的和弦进行模式（如1645、4536等）
2. 缺乏和弦替换和扩展的知识
3. 没有考虑旋律与和声的互动关系
4. 忽略调性变化和转调的可能性

**解决方案**：
1. **学习和弦替换技巧**：
   - 尝试用七和弦、九和弦替换三和弦
   - 使用转位和弦改变低音线条
   - 尝试借用和弦（Borrowed Chords）

2. **扩展和声进行**：
   - 在和弦进行中插入经过和弦
   - 使用副属和弦增加紧张感
   - 尝试调式互换（Modal Interchange）

3. **注重旋律与和声的互动**：
   - 根据旋律音符选择和弦
   - 使用对位法思维编写和声
   - 注意声部进行的流畅性

4. **打破调性限制**：
   - 尝试临时转调
   - 使用半音阶进行
   - 探索非功能和声

**相关技法**：
- 和弦替换
- 和声扩展
- 对位法
- 调性变化
