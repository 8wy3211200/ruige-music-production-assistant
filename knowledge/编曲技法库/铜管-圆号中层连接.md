# 圆号中层连接

**核心概念**：圆号是铜管组里非常关键的中间层。它既有铜管的力量，又能连接弦乐、木管和更硬的小号 / 长号。

**具体操作**：
1. 如果铜管听起来硬、突、散，先检查中层有没有圆号。
2. 小号负责亮边时，圆号可以补中层和声。
3. 长号负责重量时，圆号可以让中间不空。
4. 圆号不要永远写成主角，它最常见的价值是“接起来”。

**配器示例**：
- 小号高层做宣告。
- 圆号中层铺和声或重复旋律核心音。
- 长号 / 大号低层给根基。

**适用场景**：
- 铜管一进来很硬
- 小号和长号之间断层明显
- 高潮需要厚，但不能炸成一团

**常见误区**：
- 只有小号和长号，没有圆号过渡。
- 圆号和其他铜管同度堆满，失去中层连接作用。
- 用更多小号解决中层空的问题。
