# 铜管高潮推进

**核心概念**：铜管推进高潮，不是只把音量推大。有效的高潮来自乐器层次、八度范围和有效音区同时打开。

**具体操作**：
1. 先决定这句铜管是“宣告”“托底”还是“加重量”。
2. 第一层只加圆号或长号，不要一开始全组满堆。
3. 第二层再加小号高层亮边。
4. 第三层再扩大八度范围，低层加入长号 / 大号支撑。
5. 如果听起来糊，先减少同度堆叠，再检查低音是否太碎。

**配器示例**：
- 预备段：圆号中层长音。
- 副歌前半：小号高八度重复旋律尾音。
- 副歌后半：长号和大号托根音，打击乐给边界。

**适用场景**：
- 副歌 / 高潮不够大
- 铜管一加就假大、炸、糊
- 影视 / 管弦段落需要升级

**常见误区**：
- 一上来全铜管齐奏。
- 只推总音量，不改组织。
- 低音铜管写得过快过碎，失去根基。
