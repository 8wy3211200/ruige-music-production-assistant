# 木管高八度提亮旋律

**核心概念**：木管最常用的高价值动作之一，是在高八度给旋律加轮廓光，而不是在同音区硬堆。

**具体操作**：
1. 先让主体旋律由弦乐、人声或其他主角负责。
2. 如果旋律不够亮，用长笛或单簧管在高八度重复关键句。
3. 只重复最需要被听见的片段，不必从头到尾跟满。
4. 如果高八度太抢，降低力度或减少音符密度。

**音符 / 配器示例**：
- 小提琴旋律：A3-C4-D4-E4。
- 长笛高八度重复：A4-C5-D5-E5。
- 结果：旋律边缘更亮，但主体仍然是弦乐。

**适用场景**：
- 旋律不够亮
- 弦乐旋律被背景吞掉
- 想让段落更轻、更空、更有空气感

**常见误区**：
- 和主体同音区硬叠，导致变厚但不清楚。
- 整段都跟，导致木管没有呼吸。
- 用木管去硬顶厚铜管。
