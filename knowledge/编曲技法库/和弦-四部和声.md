# 四部和声

**核心概念**：把一个和声分成四个独立的声部来思考：低音、次中音、中音、高音

**具体操作**：
1. 将和弦的组成音分配到四个声部
2. 低音声部：和弦根音或低音进行
3. 次中音声部：和弦三音或七音
4. 中音声部：和弦五音或三音
5. 高音声部：和弦根音、三音或旋律音
6. 钢琴上用双手演奏这四个声部

**音符示例**：
- C大三和弦四部和声：
  - 低音：C2
  - 次中音：E2
  - 中音：G2
  - 高音：C3
- 排列时考虑哪些音在低音区、中音区、高音区

**适用场景**：
- 钢琴写作的基础逻辑
- 弦乐组写作的声部安排
- 任何需要完整和声表现的乐器编配
- 学习和声进行的基础方法

**实战案例**：
1. 钢琴柱式和声：按照四部和声逻辑排列和弦
2. 弦乐组写作：小提琴一提、二提、中提、大提对应四个声部
3. 和声学习：通过四部和声理解声部进行
4. 实际应用：在编曲中确保每个声部都有合理的音域和进行
