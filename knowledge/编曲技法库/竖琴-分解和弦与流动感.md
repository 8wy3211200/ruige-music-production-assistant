# 竖琴分解和弦与流动感

**核心概念**：竖琴最自然的价值是流动、空气和闪光，不是把钢琴和弦慢速复制一遍。

**具体操作**：
1. 先把和弦缩成 3-4 个核心音，不要写成厚钢琴块。
2. 用上行或下行分解制造流动。
3. 每只手不要写得过密，避免钢琴化。
4. 想要透明感时，可以和长笛、钢片琴或弱奏弦乐配合。

**音符示例**：
- Cmaj7 可以写成 C3-G3-B3-E4 的分解，而不是两手同时砸满。
- Am9 可以先保留 A-E-G-B-C 里的关键色彩音，不必每个音都塞进去。

**适用场景**：
- 过门
- 梦幻 / 透明 / 轻盈段落
- 大编制里补流动和空气感

**常见误区**：
- 把竖琴写成慢钢琴。
- 每个和弦都写得又厚又密。
- 希望竖琴承担厚重主体。
