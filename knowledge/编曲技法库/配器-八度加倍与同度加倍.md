# 八度加倍与同度加倍

**核心概念**：八度加倍更容易让旋律变清楚、变亮、变大；同度加倍更容易改变音色，让不同乐器混成一种新颜色。

**具体操作**：
1. 想让旋律更清楚、更推出去，优先试八度加倍。
2. 想要混合音色，才考虑同度加倍。
3. 如果同度加倍后变糊，先撤掉一件乐器，或改成高八度 / 低八度。
4. 高潮推进时，优先扩大八度范围，不要只堆同度。

**音符 / 配器示例**：
- 弦乐旋律在 C4-D4-E4，长笛高八度重复 C5-D5-E5，旋律会更亮。
- 双簧管和小提琴同度重复，会更快形成混合色，但双簧管个性可能被抹平。

**适用场景**：
- 旋律不够突出
- 铜管 / 木管一加就糊
- 高潮需要变大但不能变乱

**常见误区**：
- 以为同度加倍就是“更强”。
- 全部乐器同度齐奏，导致音色糊成一团。
- 只推音量，不改变八度分布。
