# 不要模仿钢琴表面手型

**核心概念**：钢琴谱里的很多写法是因为两只手、踏板和键盘限制形成的。改编成乐队时，要翻译功能，不要照抄手型。

**具体操作**：
1. 看到钢琴分解和弦，先判断它是和声、流动、还是节奏。
2. 看到钢琴厚和弦，先判断哪些音是真正必要的骨架音。
3. 看到踏板延长的声音，改成乐队持续音或分层支撑。
4. 看到快速重复音，先判断是紧张感、能量边缘，还是手法需求。

**改编示例**：
- 钢琴踏板拖长低音：乐队里可以改成低音提琴 + 大管持续。
- 钢琴分解和弦：可以交给竖琴 / 弦乐分解 / 木管流动句。
- 钢琴厚块和弦：可以拆成弦乐主体 + 圆号中层 + 木管上色。

**适用场景**：
- 钢琴谱改管弦时听起来像“插件钢琴换皮”
- 改编后很满但不自然
- 用户不知道每个音该交给谁

**常见误区**：
- 把钢琴每个音原样分给不同乐器。
- 为了“像原谱”牺牲新编制的成立方式。
- 只考虑音高，不考虑功能。
