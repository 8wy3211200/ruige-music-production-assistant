# 留空制造段落变化

**核心概念**：段落变化不一定靠加更多乐器。很多时候，在新材料进来之前留一点空白，比直接堆乐器更有效。

**具体操作**：
1. 在副歌、主题或新旋律进入前，先减少一层背景。
2. 让听众先感到“空间打开了”，再让新材料进来。
3. 如果段落一直很满，先删一层中频或高频持续音。
4. 保留低音和关键节奏，避免音乐彻底塌掉。

**配器示例**：
- 主歌结尾先撤掉 Pad 和高频小打，副歌第一拍再加入弦乐和铜管。
- 新旋律进来前，让木管停一拍，只保留低音和鼓。

**适用场景**：
- 主副歌区别不明显
- 高潮进来没有冲击
- 用户说“怎么越加越平”

**常见误区**：
- 认为段落升级只能加乐器。
- 每个空隙都想填满。
- 副歌前已经堆满，导致副歌没有再打开的空间。
