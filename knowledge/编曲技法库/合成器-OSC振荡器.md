# OSC振荡器（合成器）

**核心概念**：振荡器是合成器产生基础波形的核心部件

**具体操作**：
1. **波形选择**：
   - 正弦波（Sine）：纯净、柔和，适合sub bass和纯音
   - 锯齿波（Saw）：明亮、丰富谐波，适合lead和bass
   - 方波（Square）：空洞、类似笛声，适合bass和pluck
   - 三角波（Triangle）：介于正弦和方波之间，柔和
   - 脉冲波（Pulse）：可调宽度的方波，音色变化丰富
   - 噪声（Noise）：白噪声、粉噪声，用于打击乐和效果

2. **振荡器数量**：
   - 单振荡器：音色简单
   - 双振荡器：可创造更丰富的音色，通过失谐（detune）增加厚度
   - 多振荡器：复杂音色，可分层叠加

3. **振荡器参数**：
   - 音高（Pitch）：基础音高设置
   - 失谐（Detune）：多个振荡器之间轻微音高差异，创造合唱效果
   - 八度（Octave）：振荡器八度设置
   - 半音（Semitone）：振荡器半音偏移
   - 音分（Cent）：微调音高

4. **振荡器调制**：
   - LFO调制音高：创造颤音效果
   - 包络调制音高：创造音高变化
   - 键盘跟踪：音高随键盘位置变化

**音符示例**：
- 锯齿波lead：C4八分音符旋律，失谐+7音分增加厚度
- 方波bass：C2四分音符，脉冲宽度50%
- 正弦波sub：C1全音符，纯净低音
- 双振荡器pad：锯齿波+方波，失谐-5和+5音分

**适用场景**：
- 根据所需音色特点选择波形
- 通过振荡器组合创造复杂音色
- 调制振荡器参数增加动态变化

**实战案例**：EDM lead音色：使用两个锯齿波振荡器，振荡器1失谐-7音分，振荡器2失谐+7音分，通过LFO轻微调制音高创造颤音
