# 先抓骨架再分配乐器

**核心概念**：从钢琴、和弦或简化素材改成大编制时，不要直接照抄表面音符。先抓旋律、低音、和声和段落，再分配乐器。

**具体操作**：
1. 先标出主旋律。
2. 再标出低音根基。
3. 写出和声骨架和和声节奏。
4. 判断哪些是伴奏型，哪些是不可替代的特色动作。
5. 最后才考虑交给弦乐、木管、铜管、竖琴或打击乐。

**改编示例**：
- 钢琴左手分解低音，本质可能是低音根基 + 和声填充。
- 到乐队里可以变成低音提琴 / 大管托根音，弦乐或竖琴负责分解流动。

**适用场景**：
- 钢琴谱改管弦
- 和弦加旋律扩成完整编曲
- 简化谱改成更成熟的器乐版本

**常见误区**：
- 直接复制钢琴手型。
- 还没判断功能，就给每个音分乐器。
- 先谈音色，后谈结构。
