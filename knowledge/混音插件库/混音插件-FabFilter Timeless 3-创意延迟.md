# FabFilter Timeless 3 创意延迟

## 核心概念
- Timeless 3 是带滤波、效果和调制系统的创意延迟，不只是普通 echo。
- Delay controls 决定节奏骨架，Filters 和 Effects 决定延迟尾巴的音色身份。
- 调制槽、XLFO、Envelope Follower、MIDI Source 和 XY 可让延迟活起来。
- 实战中先把 delay time 和 feedback 管住，再添加颜色和运动。
- 发送轨适合共享空间，插入轨适合特殊声源设计。

## 使用场景
- 人声回答、吉他氛围、合成器运动、dub 式反馈、过门 FX。
- 需要清晰节奏时，低 feedback、滤掉低频。
- 需要宽而动的空间时，用滤波和调制处理 wet signal。
- 需要只在尾字出现时，自动化 send 或 mix。

## 核心参数
- 需要极端特效时，导出音频再剪，避免反馈失控。
- Delay Time：同步节拍或自由时间。
- Feedback：控制重复次数和能量积累。
- Filters：塑造延迟频谱，防止低频堆积。
- Effects：给延迟尾巴加 chorus、diffusion、drive 等性格。
- Modulation Slots：把 LFO/Envelope/XY 分配到目标。

## 实战案例
- Input/Output Mix：决定插入或发送用法。
- 主唱尾字：1/8D，低切 wet，feedback 自动化到句尾。
- 吉他氛围：较长 delay，滤暗，高 feedback 但低 mix。
- 合成器 arpeggio：tempo sync，XLFO 调 filter cutoff，制造运动。

## 补充提醒
- 过门 FX：feedback 短暂推高，导出后切掉多余尾巴。
