# SPL Transient Designer Plus 瞬态塑形

## 核心概念
- 这不是传统“先设阈值、再调 attack/release”的压缩器，而是一个电平无关的瞬态塑形工具。官方手册强调它用 `Attack` 和 `Sustain` 两个核心旋钮，直接改声音的音头和尾巴。
- 它最适合解决“音头不对、尾巴太长、节奏不利索、瞬态不够站出来”这类问题，而不是代替所有压缩和 EQ。
- `Transient Designer Plus` 比老式两旋钮版本多了内部 Sidechain Filter、外部 Sidechain、Dry/Wet 并行和软削波 Limit，所以它既能做简单塑形，也能做更聪明的定向塑形。
- 如果和现有方法逻辑对齐，可以把它理解成“比压缩器更直观地改音头音尾”的工具。用户调 Pro-C 这类压缩器越调越乱时，它往往更快出结果。

## 使用场景
- 优先用在鼓、打击乐、鼓组 room、拨弦明显的吉他、需要更清楚弹性和停顿感的 Bass。
- 优先用在“节奏明明没错，但听起来不利索”“鼓点站不起来”“低频尾巴太拖”这类问题。
- 官方手册给了很明确的例子：Kick 增强攻击感、Snare 缩短拖尾、Toms/Overheads 缩短延音、Bass 从 legato 改成更利索的 staccato。
- 也适合在不想用 Gate 的地方用负向 `Sustain` 做更音乐性的收尾控制，因为它不会像固定 release 的门限器那样切得很死。
- 不该先用在整首混音总线上。官方手册直接说 mastering 场景通常不推荐整首混音一起处理，更适合先动单个元素。
- 如果问题本质是频率堆积、谐波不对或立体声关系不对，先用 EQ、饱和或声像工具，不要把瞬态工具当万能修复器。

## 核心参数
- 如果是小白，先动两个参数：`Attack` 和 `Sustain`。
- `Attack`：正值会把音头顶出来，负值会把音头磨平。Kick、Snare、Pick Bass、拨片吉他最常用。
- `Sustain`：正值会把尾巴和房间感拉长，负值会把尾巴收短。拖尾太糊、低频不利索时特别有用。
- `SC Filter`：只让某一段频率主导瞬态检测。典型用法是别让 Kick 的大低频把整段 loop 的处理逻辑带偏，或者只针对某些击打成分加瞬态。
- `Parallel Mix`：重处理后再混回原声，比一把梭更稳。适合鼓组、Room、并行攻击感。
- `Limit`：只是在重手处理时帮你减少内部数字削波，不是真正的峰值砖墙限制器。听到输出变硬时，还是要回头减 `Attack`、`Sustain` 或 `Output Gain`。

## 实战案例
- Kick 不够顶：先把 `Attack` 往正方向推，再根据尾巴长短少量减 `Sustain`。这比用压缩器硬挤 punch 更直接。
- Snare / Toms 尾巴太拖：先减 `Sustain`，再决定要不要补一点 `Attack`。官方手册把它当成 Gate 的音乐性替代方案。
- Drum Room 不够炸：同时加 `Attack` 和 `Sustain`，再用 `Mix` 混回。手册里明确提到它能把“鞋柜房间”推成更夸张的 room 感。
- Distorted Guitar 太糊：适当减 `Sustain` 收掉失真后的拖尾，再看要不要加一点 `Attack` 让拨弦更清楚。
- Slap / Pick Bass 低频一坨：先减 `Sustain` 让低频停顿更清晰，再加一点 `Attack` 把弦击感拉出来，这和现有问答里“让人听见的是高频触感，支撑音乐的是低频”是对齐的。
- Reverb 太普通：手册里给了非常激进的玩法，用它改混响包络。实际知识库里可以保留这个思路，但不建议把它当用户第一入口。
