# bx_masterdesk Pro 一站式母带精修

## 核心概念
- `bx_masterdesk Pro` 可以理解成原版 `bx_masterdesk` 的“可控版”。核心思想没变，还是一条链里解决响度、频响、压缩和限制，但给了更多真正会影响判断的细节控制。
- 官方手册里最值得保留的升级点是：压缩段增加了 `Ratio / Release / Link / Glue / Comp Mix`，限制段增加了 `Clipper`、`Limiter Mode`、`True Peak`，同时补了更完整的 `EQ`、`HP/LP`、`De-Esser` 和立体声工具。
- 它适合“我想要一体式工作流，但又不想被原版 masterdesk 的黑盒感限制太多”的用户。
- 但它也更容易让用户乱加模块。它的价值不是参数更多，而是让你在一条链里更清楚地决定哪里在负责响度、哪里在负责胶合、哪里在负责音色。

## 使用场景
- 优先用在需要更稳定交付的流媒体母带、教学案例母带、总线精修试听版。
- 适合那些已经知道压缩、限制、EQ 各自大概在干什么，但还不想每次都从零搭整条母带链的用户。
- 适合在同一条链里同时解决“还不够响”“低频太散”“高频有点硬”“边缘太宽或太空”等复合问题。
- 不该先给完全没有母带概念的小白。那种情况原版 `bx_masterdesk` 更像训练轮。
- 如果问题本质是混音没平衡、动态编排没做好、低频彼此冲突，还是应该回混音层解决，而不是指望 Pro 版一条链补救。

## 核心参数
- 如果是小白，第一层仍然先动 `Volume` 和 `Foundation`；第二层再碰 `True Peak` 和 `Comp Mix`。
- `Volume`：继续决定整条链被推到什么程度。它不只是更响，也会牵动压缩、clipper 和 limiter 的工作量。
- `Foundation`：快速决定整体重心往低频还是上中频靠。很多时候这是比三段 EQ 更高效的第一步。
- `Comp Ratio / Release / Link`：这三个参数决定它和原版拉开差距的地方。比起只换模式，你能更明确地决定压缩是在“稳住”还是在“压扁”。
- `Glue`：给压缩段额外的总线胶合感。少量有用，过多会让瞬态往后退。
- `Clipper`：在 limiter 前先削瞬时尖峰。它能帮你更响，但也最容易把鼓、人声辅音和空气感搞得太硬。
- `Limiter Mode`：`Modern / Classic` 决定限制器的手感，`Turbo` 继续影响最终的大响度倾向。
- `True Peak`：做流媒体或任何要避开编解码后超峰值的交付时，建议优先打开。
- `Mono Maker / Stereo Width / Stereo Enhance`：这几项适合做空间重心修整，但前提永远是先保住中间主体。
- `3-Band EQ + HP/LP`：这部分该用来做收尾，不该用来替代前面更根本的动态和重心判断。

## 实战案例
- 需要更稳的流媒体母带：先用 `Volume` 和 `Foundation` 定大方向，再开 `True Peak` 控制交付边界，最后用少量 `Clipper` 换取额外响度。
- 低频肥但不结实：先收 `Mono Maker`，必要时用 `HPF` 做极低频清理，再看是不是还需要 Foundation 往上提一点。
- 一推响度就发硬：先减 `Glue` 或 `Clipper`，再放慢 `Release`，不要先拿大量高频 EQ 去“补空气”。
- 人声齿感在整体母带里被推出来：优先用内置 `De-Esser` 处理，再决定是否真要减 `Presence / Treble`。
- 如果空间越来越假：先关掉过量的 `Stereo Enhance`，确认不是宽度工具在制造“更大但更虚”的错觉。
