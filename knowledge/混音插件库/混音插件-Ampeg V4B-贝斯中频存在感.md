# Ampeg V4B 贝斯中频存在感

## 核心概念
- `Ampeg V4B` 的关键词不是单纯“大低频”，而是经典全电子管 Bass Head 的中频存在感、颗粒感和 grind。
- 官方手册第一页就把它的核心写得很明确：这是一个 100W all-tube bass head，重点是 pure tube tone 和 unmistakable grind。
- 它和另外两台 Ampeg 在知识库里的分工可以很清楚：`B15N` 更偏厚顺录音感，`SVTVR Classic` 更偏经典大箱头冲击感，`V4B` 则更适合“Bass 要从混音里站出来”的情况。
- 最关键的不是低频旋钮，而是 `1·2·3 Mid Frequency` 频点选择配合 `Midrange` 旋钮。这是它最值得被 AI 推荐时说清楚的地方。

## 使用场景
- 优先推荐给 Rock、Indie、Alt、Live Band、Pick Bass，或者“Bass 明明有低频，但在吉他墙里听不清形状”的情况。
- 优先用在需要中频存在感、拨弦颗粒、穿透力的时候。
- 也适合处理“DI 太平、太没性格，但又不是单纯想更厚”的素材。
- 不该先推荐给那种就想做特别圆、特别软、特别复古顺滑的 Bass，那通常 `B15N` 更直接。
- 如果用户的问题只是低频过多、演奏不齐、动态乱，不要跳过基础处理直接上 V4B 狂加中频。

## 核心参数
- 如果是小白，先动两个参数：`Mid Frequency` 和 `Midrange`。
- `1·2·3 Mid Frequency`：官方给了三个核心频点，`220 Hz / 800 Hz / 3 kHz`。这让它不是泛泛地“加中频”，而是能决定你要的是身体感、存在感，还是咬字边缘。
- `Midrange`：逆时针会更 contour、更远；顺时针会更 cut through。这个描述非常适合直接转成教学语言。
- `-15 dB Input`：主动 Bass、高输出拾音器、太热的 DI 先试这个，别让前端一直过载。
- `Gain / Master`：前者决定前级推进，后者决定整体输出。想要管味和 grit，通常不是只拧一个旋钮就够。
- `Ultra-Lo / Ultra-Hi`：一个补低频重量，一个补高频边缘，但 V4B 真正的主角还是中频控制，不要被两个开关带偏重点。
- `Horn`：给更多 attack tone。官方手册甚至单独为这个号角做了脉冲响应，所以它对拨片感和边缘感是有帮助的。
- `Tight / Smooth`：可以在 `Pre` 或 `Post` 工作。输入源先天脏，就用 `Pre`；成品音色想再修边，就用 `Post`。
- `Noise Gate`：只在高增益、底噪、停顿太脏时才开，而且 Threshold 别切掉自然延音。

## 实战案例
- Bass 在吉他里埋了：先把 `Mid Frequency` 放到 `800 Hz`，再把 `Midrange` 稍微往右推。很多时候这比再补低频有效。
- Pick Bass 想更有存在感：先试 `3 kHz` 的中频点，再决定 `Horn` 要不要开。这样比一味开 `Ultra-Hi` 更容易得到有信息量的攻击感。
- 想厚一点但别轰：可以先试 `220 Hz` 频点做适量中频支撑，而不是直接大加 `Bass`。
- 主动 Bass / 热前端：先按下 `-15 dB Input`，必要时再减 `Input Gain`。先控制前端，后面调音才会更稳。
- 低频太散：先试 `Tight Post`；如果是拾音器本身带进来的 rumble，就先试 `Tight Pre`。
- 官方给了很实用的建音顺序：Tone Stack 先放中间，先轮流听 `Recording Chains`，选到最接近目标的链路，再开始动 `Gain / Master / Midrange`。
