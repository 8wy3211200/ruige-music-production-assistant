# AMEK EQ 200 母带精修均衡

## 核心概念
- `AMEK EQ 200` 的核心不是“多一个 5 段 EQ”，而是把偏 `GML 8200` 思路的高精度、透明、母带级参数均衡做进了 Plugin Alliance 体系。
- 官方手册对它的描述很明确：高透明度、大 headroom、精细精修，同时补了现代工作流需要的 `HP/LP`、`M/S`、`Mono Maker`、`Stereo Width`、`THD` 等功能。
- 它更像“透明且能很细地修”的母带 / 总线 EQ，而不是用来制造大风格变化的味道机。
- 和 `AMEK EQ 250` 相比，`EQ 200` 更适合“精修、校正、平衡、母带清理”，而不是靠甜味或更明显的曲线性格取胜。

## 使用场景
- 优先用在母带、小幅总线校正、需要高透明度的混音总线精修。
- 也适合人声、钢琴、弦乐等需要“修到了但别太显”的场景。
- 优先用在“这段频率就是要更准一点、更干净一点、更平一点”，而不是“我要强烈风格化”的问题。
- 不该先用在用户只想快速提亮、快速变厚的场景，那类场景 `EQ 250`、Silver Bullet 这类更直觉。
- 如果问题是动态、失真、瞬态或立体声失衡，先别指望只靠这个 EQ 一把梭解决。

## 核心参数
- 如果是小白，先动两个参数：`Gain Scale` 和 `Auto Listen`。
- `Gain Scale`：这是它很实用的“全局 more / less”。一组 EQ 曲线大方向对了，但整体太多或太少时，不用重调每段。
- `Auto Listen`：调频率和 Q 时自动只听这一段，非常适合找问题频点和做母带小修。
- `Q`：五段都可调，低高两端还能切 shelf。既能做宽带平衡，也能做相对精细的修整。
- `Mono Maker`：母带和总线里非常实用。官方手册也直接建议常见范围在 `100-200 Hz` 一带。
- `M/S` 和 `Stereo Width`：这让它在母带上的空间修整能力很强，但一切前提是别为了“更宽”牺牲中间的稳固。
- `THD`：它不是主角，只是给过于干净的处理加一点模拟感。默认保守就够，别把它当饱和器。

## 实战案例
- 母带总线想更干净：先用 `Auto Listen` 找轻微浑浊和刺耳点，再用 `Gain Scale` 统一微调整套曲线。
- 低频边缘太散：先用 `Mono Maker` 收一下低频立体声，再判断是不是还需要 EQ。
- 侧边太亮或太空：进 `M/S` 模式单独修 Side，比全频一起削更稳。
- 人声总觉得有一层硬：先做窄一点的 Bell 微调，不要一上来就大 shelf 拉暗。
