# FabFilter Pro-Q 4 频谱动态与精修

## 核心概念
- Pro-Q 4 在 Pro-Q 3 基础上增加 spectral dynamics、instance list、EQ Sketch 和 character modes。
- Spectral dynamics 用于更细粒度地压制问题频率，不必手动建立很多窄带。
- Instance List 可在一个界面管理多个 Pro-Q 4、Pro-C 3、Pro-DS、Pro-G 实例。
- EQ Sketch 适合快速画出大方向，再微调具体频段。
- Character Mode 可给 mix 加模拟或 vintage 质感，但不要在修正 EQ 时误加颜色。

## 使用场景
- 复杂混音项目的跨轨频率管理、母带精修、人声刺耳控制、编曲密集时清理冲突。
- 需要快速试 tonal curve 时，用 EQ Sketch 打草稿。
- 需要多个轨道频率互相打架时，用 instance list 快速定位。
- 需要细碎刺耳共振时，用 spectral dynamics。

## 核心参数
- 需要透明修正时，关闭不必要 character。
- Spectral Dynamics：处理瞬时问题频率。
- Dynamic EQ Attack/Release：控制动态响应速度。
- Instance List：跨实例查看和管理。
- EQ Sketch：绘制目标 EQ 轮廓。
- Character Mode：添加模拟颜色。

## 实战案例
- Fractional Slope：精细设置高低通斜率。
- 人声刺耳：先用 spectral dynamics 控制细碎峰值，再少量 broad EQ。
- 母带倾向：EQ Sketch 画轻微 tilt，后续每段控制在小幅度。
- 多吉他冲突：instance list 对照各轨 EQ，避免重复堆高频。

## 补充提醒
- 复古总线：Character 只少量开，确认不是响度错觉。
