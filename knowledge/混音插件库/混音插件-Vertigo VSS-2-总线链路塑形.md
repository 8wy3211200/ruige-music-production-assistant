# Vertigo VSS-2 总线链路塑形

## 核心概念
- `Vertigo VSS-2` 不是单一功能插件，而是一条集成链路：`滤波 -> VCA 压缩 -> EQ -> 管饱和`。
- 官方手册最值得提炼的逻辑不是每个模块有多少钮，而是它允许你在一条总线链里快速安排“先过滤、再压缩、再 EQ、再饱和”或前后换位。
- 它特别适合那些“混音已经方向对了，但需要一条链路把密度、轮廓和颜色统一收一下”的场景。
- 不过它不是教学意义上的第一只母带工具。它更像会用链路思维的人拿来快速定型的组合器。

## 使用场景
- 优先用在 mix bus、drum bus、music bus、stem 总线上。
- 适合做总线紧致、轻压缩、EQ 微修和谐波加味的一站式处理。
- 适合那些“单独一只 compressor / EQ 解决不了，我需要一条小链路一起推”的问题。
- 不该先用在需要超透明母带精修的最后一步，也不该拿来替代手术型 EQ。
- 如果用户还分不清“是频率问题还是动态问题”，先别急着用这种串联链路。

## 核心参数
- 如果是小白，先动两个模块：`VSC-3 Compressor` 和 `VSE-2 Filter`。
- `HPF / LPF`：先把无用的低频或高频边缘清掉，再让压缩和饱和接手，常比直接压缩更稳。
- `SC Filter`：让压缩器少被低频误触发。总线一压就塌时，这通常是第一排查点。
- `Attack / Release / Auto Release`：压缩段仍是标准抓手，`Auto Release` 更适合保住音乐性。
- `Pre Comp`：把 EQ 放到压缩前。想让某个频段先影响压缩行为，再考虑这个按钮。
- `High / Low Shelf Slope`：官方说明里明确提到可切到更像 80 年代英式台子的暖曲线，这对总线甜化很关键。
- `Tube Saturator Drive / THD Mix / Shape`：决定饱和多少、混多少、以及高频失真产物要不要收掉。`Shape` 往往比单纯退 Drive 更像“去毛刺”。
- `Input Filter Type` 与 `Post Filter`：可以让饱和只作用某段频率，或调整它在滤波前后的位置。

## 实战案例
- 鼓总线一压就发闷：先用 `HPF` 清一点无用低频，再开 `SC Filter`，通常比一味放慢 attack 更有效。
- 音乐总线想更厚但不要整个都糊：在饱和段用 `Input Filter Type` 只喂低中频或上方需要的区域，不要全频一起 Drive。
- 总线已经平了，但还是缺一点前景和亮度：先试 `Pre Comp` 把 EQ 放前，让压缩对修过的信号工作，再决定是否加少量管饱和。
- 饱和一上就毛：先动 `Shape` 和 `THD Mix`，不要第一反应把 Drive 关死。
