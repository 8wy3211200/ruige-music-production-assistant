# Black Box Analog Design HG-2 电子管谐波饱和

## 核心概念
- `HG-2` 的核心不是“失真更多”，而是用串联的主饱和段加并联饱和段，让声音在不明显像压缩器工作的情况下变得更厚、更大、更有密度。
- 官方手册最值得保留的逻辑，是 `Pentode + Triode + Parallel Saturation` 这三层并不是同一种脏法。它更像在控制厚度、咬感、空气感和表观响度之间的比例。
- 它适合解决“声音太干净、太平、太像数字轨道堆起来”的问题，不适合替代真正的动态控制或频率修正。
- 对用户最重要的判断不是“这只管子是什么型号”，而是知道它一不小心就会把原本需要透明的东西做得太油、太暗、太挤。

## 使用场景
- 优先用在总线、母带、人声、贝斯、合成器等需要更多厚度和存在感的场景。
- 适合那些“不是单纯要更亮或更响，而是想更饱满、更有肉、更有模拟感”的问题。
- 适合在压缩之后做音色补偿，也适合在限制前少量增加密度。
- 不该先用在已经很拥挤、很糊、低频很多的素材上。那会让判断更差。
- 也不该拿它修刺耳、修共振、修瞬态，这些都不是它的第一职责。

## 核心参数
- 如果是小白，先动两个参数：`Density` 和 `Mix`。
- `Density`：最值得记住。它会把主要电子管段推得更深，同时做输出补偿，所以很适合边听边找“刚开始有味道，但还没失控”的点。
- `Pentode`：更像主体厚度和能量感的来源，先决定大方向。
- `Triode`：更容易带来颗粒、边缘感和存在感，过量时也最容易显脏。
- `Saturation` 与 `Sat Frequency`：并联饱和可决定重点去影响低频、全频还是高频。想让贝斯更托底和让上方更咬，选择会完全不同。
- `Calibration`：`Dark / Norm / Bright` 决定整体倾向。很多“是不是太暗”与其大改 EQ，不如先看校准模式站没站对。
- `Air`：高频空气感入口，但它应该是甜味剂，不该拿来掩盖整体已经被做浑的问题。
- `Mix`：总线和母带上非常关键。HG-2 很容易一开始觉得“更大更爽”，回混一点通常更稳。

## 实战案例
- 总线太干净、像没粘住：先加少量 `Density`，再用 `Mix` 回一点，通常比盲目加总线压缩更自然。
- 人声想更靠前但不想单纯加亮：先加轻微 `Pentode`，再少量 `Air`，比高架 shelf 更不容易刺。
- 贝斯想更有身体感：把 `Sat Frequency` 偏向低频区域，再用并联思路补密度，避免全频一起糊。
- 一开就觉得“更大”，但关掉再开发现细节没了：优先退 `Triode` 或减 `Density`，不要把短暂的更响误判成更好。
