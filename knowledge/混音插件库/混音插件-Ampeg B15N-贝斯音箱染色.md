# Ampeg B15N 贝斯音箱染色

## 核心概念
- `Ampeg B15N` 这类插件的价值，不是“给 Bass 随便套个箱头”，而是把干巴巴的 DI 变成更像录过箱体、录过链路的成品音色。
- 官方手册对它的定位非常明确：这是一个经典 studio bass amp 的还原，重点是厚、顺、圆、能从干净到更暖更糙一些的复古风格。
- 这版插件的真正特色不是只有一个前级，而是把 `1964 B-15NC` 和 `1966 B-15NF` 两套通道思路，以及 `Bias` 选择，揉进了一个插件里。
- 对真实使用最重要的不是死抠每个旋钮，而是先确定“哪套通道 / 哪条 Recording Chain / 要不要再用 Tight / Smooth 过滤”。

## 使用场景
- 优先推荐给手感干、太直、太平的 DI Bass，尤其是想要更像录过箱头、低频更有身体感的时候。
- 优先用在复古、Soul、R&B、City Pop、顺滑 Pop Bass、Finger Bass 这类不需要特别凶狠咬字的场景。
- 也适合给“低频有了，但音色没有故事”的素材补箱体感和录音链路感。
- 不该先推荐给特别追求现代金属感、极强中频切穿或特别锋利点击感的 Bass，这时候 `V4B` 或更 aggressive 的箱头通常更合适。
- 如果用户的问题本质是演奏不整齐、低频占位乱、动态失控，先处理演奏和动态，再上箱头模拟，别把箱头当修一切的插件。

## 核心参数
- 如果是小白，先动两个参数：`Input Channel / Bias` 和 `Rec Chains`。
- `Input Channels`：这个插件把 1964 / 1966 两组通道和 `0 dB / -15 dB` 输入都放进来了。高输出主动 Bass 或太热的 DI，先试 `-15 dB`，别先把前端打爆。
- `Bias`：官方手册强调它允许 1964、1966 前级和偏置思路交叉组合，这意味着你可以把它当成“额外一层音色方向选择”，而不是只盯着 Bass / Treble。
- `Bass / Treble / Volume`：真正细调音色之前先把它们放中间附近，再比较不同通道和不同 Rec Chain。
- `Input Gain`：官方建议大多数标准 Bass 保持 `0`，只有单线圈太弱、或前面串了别的插件把信号推太热时再改。
- `Power Soak`：手册给了一个很实用的起点，大多数场景先试大约 `-10 dB`。它的作用不是单纯变小声，而是允许你把功放推得更工作，再把音量压回可用范围。
- `Tight / Smooth`：这是非常实战的两个清理工具。`Tight Pre` 先切进箱头前的低频轰隆，`Smooth Pre` 先压输入噪声和毛刺；`Post` 则更像后端修整。
- `Rec Chains`：Brainworx 这类 Ampeg 插件里，录音链选择非常关键。官方建议也是先多试链路，再回头微调音色旋钮。

## 实战案例
- 想把 DI 变成更像成品的录音 Bass：先把 `Bass / Treble` 放中间，轮流试听 `Rec Chains`，先找到最接近目标的箱体和录音链，再回头修 Tone。
- 被动 Bass 太干太直：先试 `0 dB` 输入和不同通道，再看 `Bias` 哪个更顺。别一上来狂加低频，先决定前级底色。
- 主动 Bass / 热输入太冲：先试 `-15 dB` 输入或减 `Input Gain`，让箱头前端回到更合理的工作区间。
- 低频糊、琴体共鸣太多：先上 `Tight`，需要更自然就从 `Pre` 开始；如果只是成品音色下半身太肥，再试 `Post`。
- 高频毛、指噪刺：先试 `Smooth Pre`，因为这能在进前级之前先让信号没那么炸；如果只是成品太亮，再用 `Smooth Post`。
- 想用自己外部 IR 或真实箱体：官方手册给了 `CABINET BYPASS`，这时可以把它当箱头和前端色彩器来用。
