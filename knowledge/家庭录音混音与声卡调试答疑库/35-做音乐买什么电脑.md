# 做音乐买什么电脑

适用问题：
- 做音乐买什么电脑
- Windows 还是 Mac
- 买电脑最该看什么配置

核心结论：
- Windows 和 Mac 都能做音乐，没有绝对高低。
- 真正该优先看的，是 CPU 和内存，不是显卡。
- 如果你已经有能用的电脑，通常先用起来比继续纠结更重要。

怎么理解 Windows 和 Mac：
- Windows：性价比通常更高，软件资源和装机自由度更强。
- Mac：系统和音频驱动稳定性通常更好，尤其是只做编曲或混音时体验不错。
- 已经有哪边，就先把哪边用起来，不要为了“开始”再折腾一轮大换机。

买电脑优先看什么：
- CPU：别太弱，关系到工程和插件能不能带得动。
- 内存：至少 16G，更复杂的工程尽量往上走。
- 硬盘：系统和工程盘尽量别太小，样本库多时更明显。

不需要过度纠结的部分：
- 独立显卡通常不是重点。
- 不是越贵越专业。
- 不是参数拉满才有资格开始学。

一句话总结：
做音乐买电脑，先看 CPU 和内存；Windows、Mac 都行，稳定能干活就是当前适合你的电脑。
