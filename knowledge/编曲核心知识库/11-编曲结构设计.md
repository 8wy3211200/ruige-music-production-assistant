# 编曲结构设计

## 一、常见歌曲结构

### 1.1 基本段落概念

歌曲由不同的"段落"组成。搞清楚段落的目的是：**知道听众现在听到哪了、接下来该往哪走**。

#### 三大段 vs 分段落
- 一首歌可以分为"三段"：第一段（主歌1 + 预副歌 + 副歌1）、第二段（主歌2 + 预副歌 + 副歌2）、第三段（间奏/Bridge + 最后一次副歌 + 尾奏）。这里的"三段"是结构上的大块。
- **分段落**是将每一段内的"主歌（Verse）"、"预副歌（Pre-Chorus）"、"副歌（Chorus）"等各自划分开，方便后期对每个段落单独处理。

#### 段落命名标准

| 段落 | 缩写 | 说明 |
|------|------|------|
| 前奏 | Intro / I | 歌曲开头，通常 4/8/16 小节 |
| 主歌 | Verse / V | 叙述性段落，承载歌词主体内容 |
| 预副歌 | Pre-Chorus / Pre / PC | 主歌到副歌的过渡，起连接作用 |
| 副歌 | Chorus / C | 歌曲核心记忆点，通常情绪最高 |
| 间奏 | Bridge / B | 第三段中的转折段落，提供新鲜感 |
| 连接段 | Interlude | 较短、偏氛围的过渡段落 |
| 尾奏 | Outro / O | 歌曲结尾 |
| 渐出 | Fade Out | 音量渐弱到尾 |

---

### 1.2 三段式歌曲结构

三段式是流行音乐最主流的框架，以四个小节的段落单元来组织。

#### 结构模板

```
[Intro] → [Verse 1] → [Pre-Chorus] → [Chorus 1] → [Interlude]
→ [Verse 2] → [Pre-Chorus] → [Chorus 2] → [Interlude]
→ [Bridge] → [Chorus 3] → [Outro]
```

#### 每段长度参考
- **Intro**: 4 ~ 8 小节
- **Verse**: 8 ~ 16 小节（通常是 8 或 16）
- **Pre-Chorus**: 4 ~ 8 小节
- **Chorus**: 8 ~ 16 小节
- **Bridge**: 4 ~ 8 小节
- **Outro**: 4 ~ 8 小节

#### 段落总览示例

以《A Sky Full of Stars》为例，4/4 拍，BPM=125，整体结构为：
```
Intro 8小节 → Verse 8小节 → Chorus 8小节 → Verse 8小节 → Chorus 8小节 → Bridge 8小节 → Chorus 8小节 → Outro 8小节
```
共 64 小节。

#### 核心原则

- **主歌与副歌的情绪要有明显对比**。Verse 留能量，Chorus 释放。如果 Verse 已经很满，到 Chorus 就不会有"推进感"。
- **Bridge 一定要和前面不一样**。Bridge 的作用是打破审美疲劳——如果前面副歌已经重复了两次，到第三次再来同样的东西，听众就腻了。Bridge 要在和弦、节奏、配器上给出新鲜感。
- **重复进副歌之前的 Interlude 可以短，但不要缺席**。4 小节左右的间隔为听众提供"喘息"，让下一遍副歌推出时冲击力更大。

---

### 1.3 段落标注操作（Cubase）

#### 标记轨（Marker Track）
- 快捷键 `Ctrl/Cmd + M` 添加标记
- 每个段落开头放置一个标记
- 标记颜色分类（如 Intro=灰色，Verse=蓝色，Chorus=红色，Bridge=紫色）
- 标记命名清晰：如 "Verse 1"、"Chorus 1"

#### 拍号轨（Time Signature Track）
- 用于标记拍号变化（如 4/4 → 6/8）
- 如果全曲 4/4 不变，则只需开头设一个

#### 和弦轨（Chord Track）
- 写下整曲和弦进行
- 优点：后续修改和弦时，轨上的 MIDI 块可以跟随更新
- 对编曲布局是最重要的参考线

#### 小节范围划分建议
- 不只是标记段落起始点，还要标明这个段落的结束点（范围标记）
- 做法：拖动标记使之覆盖整个段落的小节范围
- 便于跳转编辑时快速定位段落的完整范围

## 二、层次递进设计

### 2.1 层次递进的本质

层次递进是指**编曲元素从少到多、从简单到丰富的逐步展开过程**，目的是引导听众的注意力，让音乐有"向前走"的动力。核心原则是：**不要在第一段就把所有武器亮出来**。

### 2.2 要素堆叠策略

**段落——持续积累的乐器/声部清单**

```
Intro:       鼓（仅Hi-Hat/Ride）+ 氛围Pad
Verse 1:     + 钢琴/吉他（和弦）+ 主歌主旋律
Pre-Chorus:  + 贝斯（进入） + 军鼓/Clap（反拍）
Chorus 1:    + 完整套鼓 + 副歌旋律 + 背景和声/弦乐
Interlude:   去掉套鼓和贝斯，回到Pad
Verse 2:     同Verse 1，但可加一层点缀（如弦乐长音）
Pre-Chorus 2:同前，可加一层节奏点缀
Chorus 2:    同前，可加一层旋律填充
Bridge:      换一套配器（如去掉鼓，留钢琴+人声或失真吉他）
Chorus 3:    全编制（可加管乐/大合唱等最高能量）
Outro:       逐层撤掉，回到最后只剩一个核心元素
```

#### 具体技巧

- **主歌阶段（Verse）**：控制编配密度。保持开扬、留白足够。和弦乐器不宜过多，给主旋律留空间。
- **预副歌阶段（Pre-Chorus）**：加入节奏推动力。贝斯/底鼓开始有运动感，军鼓或 Clap 进入反拍，让听众感受到"要进入副歌了"。
- **副歌阶段（Chorus）**：释放全部能量。套鼓全进、背景和声/pad 填充、旋律线升高八度或加配重奏，营造段落突破感。
- **动态对比**：并不是每次进入相同段落都是一模一样的编配。第二次副歌可以比第一次更满（如加弦乐），第三次可以再升级（如加管乐、合唱团）。
- **"加一层就减一层"**：当在新段落加入新乐器时，可以考虑拿走一件已有的乐器，避免声音过密。例如 Chorus 2 加了失真吉他，就撤掉 Verse 中的钢琴，换一种织体。

### 2.3 声场拉锯

层次递进不仅是数量上的，也是**空间布局**上的。

- Verse 段：声场相对居中、靠后（混响更大、干湿比偏湿），营造"远距离感"
- Pre-Chorus 段：慢慢拉近，乐器开始变干、靠前
- Chorus 段：声场最宽、最靠前，乐器定位左右拉开、近距感强

### 2.4 常用节奏层的增值模式

| 段落 | 节奏密度 | 说明 |
|------|---------|------|
| Intro | 稀疏（四分音符或更少） | 保持呼吸感 |
| Verse | 适度（八分音符为主） | 给主题留空间 |
| Pre-Chorus | 密集（八分+十六分装饰开始进入） | 蓄势 |
| Chorus | 饱满（十六分或以上节奏层填充） | 释放能量 |
| Bridge | 变化（可以很稀疏做对比，也可切分） | 新鲜感优先 |

## 三、情绪起伏设计

### 3.1 情绪曲线的核心逻辑

情绪起伏设计的本质是**在时间轴上构造"紧张→释放"的循环**。一首歌的节奏是：从不那么紧张到越来越紧张，然后释放，再从头开始。其中"紧张"与"释放"的幅度差就是情绪价值所在。

### 3.2 制造紧张的常用手段

**紧张 = 不稳定 / 未完成 / 积累**

| 手段 | 具体做法 | 效果 |
|------|---------|------|
| 和声紧张 | 使用属和弦、减和弦、变化和弦 | 听众感到"还没完" |
| 节奏收紧 | 节奏层加密（四分→八分→十六分） | 呼吸越来越急促 |
| 音域攀升 | 旋律线或贝斯线逐步升高 | 能量被向上推 |
| 密度增加 | 逐层叠加乐器（Pre-Chorus 最典型） | 空间越来越满 |
| 音色改变 | 滤波器的 cutoff 逐步打开 | 声音从暗变亮 |
| 音量渐强 | 整体或某声部（通常军鼓/镲片）渐强 | 物理上的压迫感 |

### 3.3 释放紧张的方式

**释放 = 解决 / 完成 / 新起点**

| 手段 | 具体做法 | 效果 |
|------|---------|------|
| 和声解决 | 属→主（V→I） | 最自然的释放 |
| 节奏放宽 | 进入副歌后节奏层稳定（不再加码） | 到达终点 |
| 全部能量爆发 | 从未有过的力度进入副歌 | 情绪奔涌而出 |
| 声场突然变宽 | 副歌段乐器拉开 panner | 空间得到"呼吸" |
| 旋律大跳进 | 旋律跳到高音区保持 | 情感升华感 |

### 3.4 实战中的常见陷阱

#### 陷阱 1：Verse 就已经很满了
- 症状：Verse 已经用了全套鼓、pad、吉他、钢琴，到 Chorus 时发现"还能加什么"
- 结果：段落之间没有层级差异，音乐听起来"平的"
- 解决：Verse 要有克制力，留出空间给后面的段落升级

#### 陷阱 2：只有积累没有释放
- 症状：Pre-Chorus 一直在堆叠，到了 Chorus 反而没变化
- 结果：能量只升不降，听众会疲劳
- 解决：Chorus 进入时要选择一种释放方式（不一定是"更响"，可以是突然变宽、变空，如只用鼓+人声两小节再全进）

#### 陷阱 3：Bridge 不够"桥"
- 症状：Bridge 段和之前段落没有显著区别
- 结果：Bridge 没有起到"提供新鲜感"的作用，听众已经开始走神
- 解决：Bridge 至少改变 2~3 个维度 —— 换和弦走向、换节奏型、换配器组合，甚至换调

## 四、记忆点设计

### 4.1 记忆点的定义

记忆点（Hook）是歌曲中最容易被记住、最想让人重复听的那部分。它不一定是旋律 —— 可以是一段 Riff、一个音色设计、一个节奏模式、或者一句歌词的处理方式。

### 4.2 记忆点的位置策略

**黄金位置：副歌开头 2~4 小节**
- 听众在副歌进入后前几秒决定"这是不是一首好歌"
- 副歌的第一句/第一个乐句应当是最抓耳的

**次要位置：**
- Intro 结尾进入 Verse 的那一下（预示"要开始了"）
- Bridge 的转折时刻（提供"啊，原来还能这样"的惊喜）
- Outro 的收尾处理（留给听众的"余韵"）

### 4.3 构建记忆点的常见技术

- **旋律性 Riff**：副歌的器乐加花或旋律线，独立于主旋律还能被记住（如《Seven Nation Army》的贝斯线）
- **节奏性 Hook**：特殊的节奏型或鼓的 Fill（如《We Will Rock You》的 stomp-stomp-clap）
- **音色性 Hook**：某个音色经过特殊处理，一听就让人识别（如滤波哇音、反向音效、失真贝斯）
- **和声性 Hook**：非预期但不突兀的和弦进行（如《Let It Be》的 IV-V-iii-IV）
- **留白性 Hook**：在所有乐器都响时突然集体停顿一拍，再炸出来

### 4.4 记忆点的重复策略

- 副歌中的核心记忆素材至少重复 2~3 次才足以被听众记住
- 但不要在每次重复时完全一样——第 2 次加一层加花，第 3 次升级配器
- 间隔策略：第一次出现后，不要马上再出现一次，让听众"想听但没听到"，期待感加重后再出现，记忆效果更好

## 五、实战案例分析

### 5.1 案例一：《A Sky Full of Stars》（Coldplay）

**结构概览**（4/4, 125BPM, 64小节）：

```
Intro (8) → Verse (8) → Chorus (8) → Verse (8) → Chorus (8) → Bridge (8) → Chorus (8) → Outro (8)
```

**层次递进分析**：
- Intro：钢琴琶音 + 简单节奏（高频镲片），氛围弱
- Verse：人声 + 钢琴 + 贝斯进入，节奏稀疏
- Chorus：套鼓全进 + 副歌旋律 + 背景和声，能量释放
- Bridge：和弦走向变化（F#m → E → D → A 换成 Bm → A → G → A 或类似），配器变薄（钢琴为主），打破循环
- Outro：逐步撤掉，回到钢琴

**记忆点**：
- Intro 的四音符钢琴琶音就是全曲最标志性的 Hook
- 副歌中的人声短句重复设计

### 5.2 案例二：《Yellow》（Coldplay）

**结构概览**（4/4, 约 87BPM）：

```
Intro (4) → Verse 1 (16) → Chorus (8) → Verse 2 (16) → Chorus (8) → Bridge (8) → Outro (8)
```

**层次递进分析**：
- Intro：只有吉他分解和弦 + 轻微背景噪音
- Verse 1：人声进入，吉他保持，非常干净
- Chorus：鼓全进 + 贝斯 + 失真吉他和弦墙，音量跃升明显
- Verse 2：加了一轨副旋律吉他，比 Verse 1 稍微厚
- Chorus 2：同 Chorus 1，但人声多了层叠
- Bridge：鼓撤掉，吉他回到分解，人声变得静谧——对比极强
- Outro：吉他扫弦 + 人声即兴，逐渐减弱

**关键启发**：
- Verse 可以长达 16 小节而不无聊，因为和弦进行有细微变化
- Outro 的处理方式是"不结束在完整副歌，而是散开"，有意犹未尽的效果
- Bridge 的"安静"作为高潮后的"降落"，营造非常强的对比

### 5.3 案例三：《Viva La Vida》（Coldplay）

**结构概览**（4/4, 约 138BPM）：

```
Intro (2) → Verse 1 (8) → Chorus (8) → Verse 2 (8) → Chorus (8) → Bridge (8) → Chorus (8) → Outro (4)
```

**入门后立刻讲故事**：只有 2 小节的 Intro，字符串乐句直接引入，没有慢慢铺垫就进入 Verse。

**Bridge 的设计**：
- 和弦进行换了（C → D → G → Em → C → D → G → G），不同于 Verse/Chorus 的循环
- 加了新的节奏元素（类似 Bell 的声音）
- 唱腔从叙述变喊唱
- 真正做到了 Bridge 的功能：提供新鲜感，避免 Chorus 重复到腻

**层次递进亮点**：
- 整曲几乎没有什么鼓但能量感充沛，靠的是弦乐组的弓法变化（长弓 vs 拨弦）、人声层数的叠加、节奏层的加密（Pre-Chorus 加 Bell）

## 附录：段落检查清单

在编曲完成后，逐项自检：

- [ ] Intro 足够引人吗？（通常 4~8 小节）
- [ ] Verse 和 Chorus 有明显的能量/密度差距吗？
- [ ] Pre-Chorus 有明确的"推进感"吗？（通常在进入 Chorus 前 4 小节开始堆叠）
- [ ] Chorus 在第一次出现后就足够有记忆点吗？
- [ ] 第二次 Chorus 比第一次更满/更有层次吗？
- [ ] Bridge 与之前的段落有显著差异吗？（至少改变 2~3 个维度）
- [ ] 第三次 Chorus 是全曲最饱满的时刻吗？
- [ ] Outro 有合适的收束感吗？（不要突然断掉，除非是设计效果）
- [ ] 在 Cubase 中：标记轨是否标注了所有段落的起止范围？拍号轨和和弦轨是否已设好？
