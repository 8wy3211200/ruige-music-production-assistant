# 其他乐器与实战案例核心知识

## 第一部分：其他乐器

### 1. 民乐

#### 1.1 古琴（古筝泛音/点状乐器）

- 在工行私人银行宣传片中，古琴泛音作为"点状"乐器使用，与二胡长线条形成"点与线"的配合关系
- 古琴泛音插入时机：跟随画面中"太阳亮了一下"的时刻，起到强调作用
- 古筝泛音还落在"组织超万人"图标出现的位置，配合画面做卡点

#### 1.2 二胡（长线条/Lead乐器）

- 二胡作为长线条乐器，与古琴点状泛音形成"线和点"的均衡配合
- 在林篇章中，二胡作为lead引领段落，配合画面"老者往上看抚摸树"开始进入
- **氛围设计中的二胡用法**：
  - 作品《归岸》中，二胡加大提琴（大提），混响开得非常大，作为solo代表"海边的孤独故事"的意象
  - 二胡负责旋律动机（主要情感载体），大海环境声负责铺底（场景氛围）
  - 通过空间处理，让二胡听起来像是在空旷的海边拉响
  - 技法：把二胡做得"非常虚"，作为音效+乐句的综合体，与海浪、时钟等环境音效结合
  - 使用逻辑：从客户沟通中提取意象（"他老听见那边有人去拉二胡"），将二胡作为情感载体融入氛围设计

#### 1.3 笛子/箫（气声/音效化）

- 在工行宣传片中，笛子的气声作为"既是音符又是音效"的手法使用
- 配合画面"树林出来同时有风存在"的时刻，增加氛围感
- 林篇章中，竹笛作为主要色彩乐器之一，配合画面山林烟雾缭绕的氛围

#### 1.4 编钟（金石之声）

- 在这个配乐中用于川主题的段落，一出场就能让人联想到中华文明的源远流长
- 编钟音色的特点：金属声偏大、乐声偏小，采样本身会有些"不准"
- **编钟的搭配技巧**：需要把古筝和编钟加在一起做double，用古筝增添乐声，让编钟声音更好地融入混音
- 单独使用编钟容易被其他乐器盖掉，需要古筝陪衬

#### 1.5 箫（林篇章色彩乐器）

- 林篇章中箫作为主要色彩乐器之一

#### 1.6 木鱼/民族打击乐（色彩点缀）

- 林篇章中木鱼在镜头切换的地方敲击，增强国风色彩
- 氛围设计中用太鼓作为主要展现乐器，base drum作为重音增强

#### 1.7 伽倻琴（朝鲜族民族乐器）——游乐园主题曲

- 伽倻琴类似古筝，演奏方法为半倚附在身上，用类似三味线的拨片演奏，音色比古筝闷
- 伽倻琴音源：Kontakt自带"亚洲"系列中的伽倻琴
- 组合技法：
  - 伽倻琴本身不够亮 → 叠加伽倻琴泛音
  - 叠加泛音还不够突出 → 再叠加扬琴（镶高频边）
  - 再叠加钢片琴 → 整体既有高频亮度又有民族音色和梦幻感
- **竖琴替代古筝的逻辑**：不希望过早把古筝音色亮出来，因为后面民乐声部有其他类似音色，过早出现会减少惊喜感

#### 1.8 民族打击乐组（游乐园主题曲）

- 前奏用带有民族特色的打击乐组，包括太鼓（重鼓点）、小鼓（轻鼓点）、民族特色的铙钹
- 节奏型参考：2002年日韩世界杯主题曲1分37秒开始的鼓点
- 用音源自带预置中的"精灵古怪"音色（钉钉、弹簧声、玩具声）添色彩
- 牛铃营造欢快感，clap在狂欢节乐曲中重要（人拍手的感觉）

#### 1.9 民乐SOLO编写相关建议

- 流行音乐和纯音乐中需要大量国风乐器：古筝、笛子、琵琶、唢呐、曲笛
- 推荐国风音源：
  - Ample Sound（Ample本身也出了国风音色）
  - 三体（3体）——唢呐非常好用
  - Kontakt自带的东方艺术家系列（East Asia / 东方艺术家）——古筝、笛子、曲笛都非常够用，使用逻辑简单

#### 1.10 氛围设计中民乐的运用

- **元素组合案例**：二胡采样 + 大海环境声FX = 电影感的建立
  - 二胡负责旋律动机（主要情感载体）
  - 大海声负责铺底（场景氛围）
  - 通过空间处理让二胡听起来像是在空旷的海边拉响
  - 瞬间建立叙事感——听众一听就知道"这是一个在海边的孤独故事"
- FX素材选择范围：环境声（海浪、风、雨）、具体物体声（心跳、马蹄、车流）、乐器采样（马头琴、二胡、钟声）

### 2. 采样与律动写作方法论

#### 2.1 鼓组采样的"真鼓思维"延伸

- 编采样鼓时，可以借鉴做真鼓的手法——做一个带嗨嗨的loop，把低频切掉放在鼓组中做高频部分
- 这样整个鼓组的音色会更整体更饱满

#### 2.2 侧链压缩在节奏音乐中的应用

- 用最重要的鼓组（地鼓+军鼓）给其他乐器整个大组做侧链压缩
- 每一下鼓打下来时，乐器组做避让，整个律动更明显
- 适用于节奏为主的音乐，尤其是广告配乐

#### 2.3 808贝斯平行失真手法

- 分三轨处理贝斯：
  - Hi轨：切低频，保留高频，可加宽
  - Low轨：切高频，保留低频，加失真/饱和（让贝斯变顶）
  - Clean轨：原声
- 三轨合并到总线上，加OTT多段压缩，最后用utility控制音量
- 存成效果链，适用于需要"很顶很结实"的电子贝斯

#### 2.4 SFX音色在律动段落的作用

- riser/downer/反转音效在段落过渡中的作用（承上启下），常用在切换镜头的位置
- 电子质感音乐中用滤波噪音/音量变化的噪音音色塑造上升下降感

## 第二部分：实战案例完整分析

### 案例1：中国风史诗管弦（工行私人银行宣传片配乐）

#### 客户需求
1. 音乐迎合广告主题"山川林"——三段分别设计音乐
2. 大气内敛、激昂磅礴（符合工商银行私人银行气质）
3. 融入国风元素
4. 尽量卡点

#### 核心设计思路

**和声设计：**
- 序章（山）：G小调1637（降B大调6415），小调色彩，责任感、宿命感
- 川段落：G小调转G大调（平行大小调转换），避免远关系调带来的突兀
- 林段落：G小调，走645循环（中国风色彩）

**开篇手法：**
- 钢琴（Keyscape中央大钢琴）+ 混响开大 → 无需pad就营造大声场
- 柱式和弦尽量简单，最后两拍加过渡音连接下一个和声
- 加小提琴泛音 → 为钢琴高频增添色彩，营造"希望感"

**配器递进层次（"不要一开始把乐器设计太丰富"）：**
1. 单纯钢琴（混响大）
2. 第二遍循环：古琴泛音（点）+ 二胡长线条（线）+ 笛子气声 → 点线配合
3. 弦乐分部和弦乐技法变化（一体点状/二体中体线状/大体点状等）
4. 木管声部加入（高音加强 + 弦乐铜管粘合剂）
5. 打击乐：军鼓（进行曲感觉）+ 太鼓（国风调性）+ 低音大鼓（增强重音）+ 过镲
6. 圆号独奏 → 凯旋般铜管音色 → 最高潮

**弦乐编写要点：**
- 新手常见错误：所有弦乐同一时间演奏相同技法
- 正确做法：点和线配合——一体点状（快速）+ 二体中体线状 + 大体点状（和弦内音）+ bass线状
- 弦乐框架=公交车的扶手栏杆，牢牢固定和声框架

**鼓组设计：**
- 军鼓：打小提类似的律动，行进的感觉
- 太鼓音色（代替传统bass drum）——更符合国风调性
- 在重拍上叠加低音大鼓增强冲击力
- 过镲作为连接手法

**调性转换技巧（川段落）：**
- 大段落转换时两点注意：①不延续前面和声（避免无聊）②不做太大调性差别（避免别扭）
- G小调转G大调——常用手法
- 换了钢琴音色（柔软音色描述水的感觉），律动改为叮咚像水流
- 小提琴泛音是"万金油"——任何低能量段落都可以用

**林段落的民乐处理（国风元素最浓的段落）：**
- 钢琴简单和声 + pad（漂浮在树林上方的雾气感）
- 竹笛和古筝为主角
- 木鱼在镜头切换处敲击
- 二胡长线条作为lead引领
- 太鼓为主要打击乐

**转场技巧：**
- 风铃转场——配乐中的"大杀器"、"万金油"
- 竖琴搭配钢琴用于清新段（不同于纯钢琴，类似古典吉他）

**高潮段落（最后辉煌段落）：**
- 铜管高度参与，民乐暂时拿掉（民乐整体表达较委婉，不适合辉煌段落）
- 木管的作用：①短笛为弦乐高音镶边增加金属感 ②巴松连接低音弦乐和低音铜管
- 最后logo落地：古琴古筝勾弦 + 打击乐重音 → 大气稳重落地

#### 修改经验
- 第一版到最终版修改不多（未涉及剧情线顺序调整，只是删减片段）
- 导演觉得林篇章中国风太重 → 直接拿掉国风章节，重写保持前后进行曲风格
- 养成好习惯：标记点分段 + 标注速度 + 写上和弦信息 → 无论怎么调整心中都有数

### 案例2：DancePop/广告配乐（蒙牛随变X说唱巅峰广告配乐）

#### 客户需求
- 时长15秒广告配乐
- 前后要有反差
- 前面old school hip hop风格
- 后面新潮电子风格

#### 核心经验：翻译客户需求
- 客户描述的"old school"和"new beat"并不准确 → 核心需求是"反差"
- 不较真风格词汇，从描述中感受真实需求
- 主动找参考方向给客户确认

#### 第一版拆解（前半段Old School部分）

**分组思路：** Basic（鼓+贝斯）+ Top（旋律乐器）+ HF（高频）

**鼓组设计：**
- BPM做到105（比通常hippop的快，为了卡时间线）
- cake + snare + 两组鼓采样groove（起到overhead的作用，只保留高频，低频切掉）
- 真鼓思维延伸：用带嗨嗨的loop做高频部分 → 鼓组音色更饱满

**顶部乐器（Top组）：**
- 铜管 + funk吉他 + 搓碟音效 + pad + 人声切片 + 小号
- 声场对称：搓碟和吉他分居左右，两者结构类似（短音色）→ 声场更宽更平衡
- pad切掉中间低频拉两边高频，吉他偏中间 → 声场对比（pad包围感 vs 吉他集中）

**音色质感：**
- RC20效果器 → 复古做旧磁带质感（noise/失真/wah等选项控制好量）
- Top组和HF组加侧链压缩，用鼓组做避让 → 律动更明显

**过渡：** 搓碟音效（倒放效果）连接两段音乐

#### 第一版拆解（后半段Future Beats部分）

**贝斯处理：** 808平行失真三轨法（Hi+Low+Clean）→ 让贝斯变得"很顶很结实"

**和弦音色：**
- 加宽 + OTT + 让和弦变得宽亮
- 和弦放在basic组（作为基础的一部分）→ 分组逻辑自洽即可

**声场对比关系：**
- pluck（顶、实）+ bell（虚、弱）→ 纵向对比，一个在前一个在后
- 一个音色偏真实 + 叠一个电子音色 → 电子音乐中得到有质感的声音（真实乐器不搭、太电子的塑料，两者结合刚好）

#### 第二版/第三版迭代

- 第二版：更反差的future bass感，铜管号角+电子号角叠层
- 第三版：完全抛弃hip hop关联，做轻柔无鼓版本（电钢琴+bell+pad+贝斯）
- **核心：不断抓取客户真实需求，翻译客户要求**
- **职业建议：** 适当向客户表明工作精力的成本，但修改是工作必备流程

### 案例3：广告配乐（劲酒广告）

#### 核心经验
- 时间极紧（一个晚上完成两分钟配乐）
- 乐器实录要求 → 如何用MIDI高效替代
- 导演给了参考音轨，但后来又推翻了音乐风格

**参考音轨分析：**
- 从参考音频提炼四大元素：轻快有力的鼓点 + 弦乐铜管（好莱坞大乐队感）+ 口哨 + 黑人风格吟唱
- 都市感、快节奏、欣欣向荣的氛围

**第一版设计：**
- 钢琴独奏开篇 + 电话音效果
- 鼓点逐步递进（前奏稀松 → 小哥打招呼后律动起来 → 送货上楼下楼高潮）
- 上方和声乐器：爵士钢琴 + Organ + 钢片琴 + 电钢琴乐句
- 人声：用路谱采样做音乐"裁缝"
- 铜管+弦乐：Session铜管/Session弦乐系列

**第二版修改（风格大转变：从都市→朋克）：**

**不改的三点：**
1. 第一段钢琴独奏段落（不影响风格的纯情绪描述）✓
2. 和声调性（和声不是决定风格的关键）✓
3. 打击乐声部和律动元素（保留拍手元素贯穿）✓

**改的三点：**
1. 取消所有乐器实录 → 全用MIDI（广告可能多次修改剪辑，实录无法灵活调整速度）
2. 电吉他成为主角（Apple吉他 + Bias外挂效果器，加载Green Day预制）
3. 用音效替代复杂乐器转场（钢琴→摩托呼啸音效→鼓花→起飞感）

**常用转场手法：**
- 铜管连音 + 吉他刮弦向下
- 音效取代乐器：钢琴→效果音+鼓花
- 拍手贯穿始终作为"音乐锚点"

### 案例4：游乐园主题曲制作（延吉恐龙王国的主题曲）

#### 设计策略
- 从商业属性出发，从游乐园属性提炼核心词汇：节庆、狂欢
- 融合当地民族文化符号（延吉→朝鲜族）

#### 前奏设计
- 民族打击乐开场（太鼓+小鼓+民族铙钹）→ 参考02年世界杯日韩世界杯主题曲鼓点
- 弦乐震音 + 竖琴（代替古筝做分解和弦，避免过早暴露古筝音色）
- 定音鼓过鼓 + 镲片过渡

#### 主旋律层设计
- 采用朝鲜族民歌《阿里郎》旋律
- 旋律层分三层：
  - 伽倻琴（主旋律，音色闷）→ 叠加伽倻琴泛音 → 再叠加扬琴（镶高频边）→ 再叠加钢片琴（梦幻感）
  - 人声四部和声（女高/女中/男中/男低）→ 注意留气口
  - 弦乐：小提琴1走阿里郎旋律（用长线条串联点状民乐）
- 竖琴弹奏分解和弦，增加灵动感

#### 节奏转换
- 曲风从86拍大气体→44拍130bpm电子律动
- 通鼓（多打鼓边，模仿狂欢节游行时背在身上的感觉）+ 牛铃 + clap
- 铜管声部（Session铜管）→ 营造节日感
- 副歌前降速降情绪做反差

#### 编曲注意事项
- 打击乐是这类歌曲的核心（打击乐没起来，上方和声写得再花哨都没用）
- 最好有占位人声，否则容易编太满 → 人声没位置
- 间奏复现阿里郎但改44拍（不能像前奏用不同节拍，要保证歌曲统一性）
- 最后一遍副歌加民乐（伽倻琴+扬琴）做pluck织体，不要太密

### 案例5：氛围设计中乐器的运用

#### 《归岸》案例分析
- **编曲协作手法**：二胡采样 + 大海环境声FX
- 二胡负责主要旋律动机（情感载体），大海声铺底（场景氛围）
- 空间处理让二胡听起来像是在空旷的海边拉响
- 联合其他元素：钢琴模仿时钟（滴答声），竖琴点状拨奏
- 最后用钢琴反相（反向音频）营造时光回溯氛围

#### 用导演思维设计Pad
- 从歌词中找意象，翻译成声音
- 环境声（海浪、风、雨）、具体物体声（心跳、马蹄、车流）、乐器采样（马头琴、二胡、钟声）
- ADSR调节让音色"呼吸"

## 第三部分：编曲心得与职业建议

### 1. 翻译客户需求
- 客户往往不是音乐专业人士，用情感词汇描述音乐 → 从词汇中感受需求，不较真风格术语
- 有参考音轨 → 大大降低沟通成本
- 没有参考音轨 → 主动找参考给客户确认

### 2. 配器设计原则
- **配器递进**：不要一开始设计太丰富，随着影片情绪一层一层烘托（像上菜先上前菜）
- **点和线配合**：任何声部（民乐/弦乐）都要遵循"有长线条有点状"均衡
- **声场平衡**：结构类似的音色在声场中对称（左右/前后/宽窄）

### 3. 修改心态
- 修改是工作必备流程（参考坂本龙一给末代皇帝配乐的经历）
- 不改的部分：①不影响风格的纯情绪段落 ②和声调性 ③打击乐律动骨架
- 大改时保留这三者 → 工作效率大大提升

### 4. 职业习惯
- 用标记点给音乐分段 → 心中清楚每块音乐的对应位置
- 标注速度 → 方便后续调整卡点
- 标注和弦信息 → 编写时"不会把音写错"

### 5. 接单与沟通
- 修改次数难以量化，但在修改中可以让对方清楚修改原因
- 如果是对方需求发生变化，可以合理提出费用要求
- 解决问题的态度比编曲技术更重要——"你是一个平衡者的角色"

### 6. 编歌型曲子 vs 配乐型曲子的区别
- 歌曲类要注意：占位人声的用途（阻止编太满）
- 音乐如果有歌词，人声需要空间——频段上留位置
- 间奏不能采用跟歌曲本身节奏不同的节拍/律动，要保证歌曲统一性
- 配乐则可以随着镜头切换随意调速度、调情绪、换风格
