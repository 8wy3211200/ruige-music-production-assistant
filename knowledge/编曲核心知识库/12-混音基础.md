# 混音基础

## 一、混音本质

### 1.1 混音到底是什么？

**混音不是让每个乐器都好听，而是让所有乐器在一起好听**。混音工作的对象从来不是单个乐器——而是乐器与乐器之间的**关系**。

混音的三维度框架（压缩版）：
1. **频率**——各乐器在频段上是否打架，有没有各自该待的位置
2. **动态**——强音和弱音之间的差距是否可控，整体音量是否稳定
3. **声场**——各乐器在立体声场中的前后左右位置是否清晰、有层次

任何时候遇到混音问题，都可以从这三个维度中选择一个去解决。不需要同时处理三个。

### 1.2 两个维度：前后 & 左右

混音的声场关系可以从两个维度理解：

- **横向维度（左右）**：Pan（声像），确定乐器在左右声道之间的位置。通常低频乐器（底鼓、贝斯）居中，中高频乐器可左右拉开。
- **纵向维度（前后）**：靠混响量和干湿比调节。混响大、干声少的乐器听起来靠后；混响小、干声多的乐器听起来靠前。

**前后与左右是两个独立的控制轴**，可以组合使用：人声通常在正前方（居中 + 干声多），打击乐可以分布在左右两侧加上不同混响营造深度层次。

### 1.3 最重要的一件事：监听环境

在开始任何混音之前，如果监听环境有问题，你做的一切都是错的。

- 音箱摆放：左音箱到右音箱的距离 ≈ 左音箱到你的耳朵的距离，三个人形成一个等边三角形
- 监听音箱的高频单元对准人耳高度
- 房间内不要完全吸音（完全吸音会让混音做出过亮的结果），也不要完全反射（会产生驻波干扰）
- 耳机只能作为参考，不能作为混音主力——耳机无法准确判断声场

## 二、混音工作流程

### 2.1 大象装冰箱三步法

混音的整体流程没有想象中复杂，可以拆成三步：

1. **第一步：听素材，确定关系**
   - 先完整听一遍所有分轨，不碰任何推子和旋钮
   - 确定：谁是主角？谁是配角？哪个乐器需要突出？哪个乐器需要往后躲？
   - 在听的过程中要对整首歌的声音画面有一个预期

2. **第二步：处理关系——解决矛盾**
   - 各乐器在频率上有冲突吗？（EQ 安排位置）
   - 各乐器在动态上有冲突吗？（压缩器控制出位）
   - 各乐器在声场上有冲突吗？（Pan 拉开 + 混响安排远近）
   - 这一阶段的目标是让所有乐器不打架

3. **第三步：让它们融合在一起**
   - 给整首歌一个统一的空间感（混响母线/Reverb Bus）
   - 总线处理（Bus Compressor / Limiter）
   - 让乐器和乐器之间听起来像一个整体，而不是各自为政

### 2.2 混音三阶段：构思到操作到复盘

- **构思阶段（最重要）**：花时间在脑海中构建这首歌应该听起来什么样。不做任何操作，只做决定。
- **操作阶段**：把构思阶段的决定逐条执行。这里考验的是**工具熟练度**——是否能快速找到需要的插件并调出想要的声音。
- **复盘阶段**：混完初稿后，不要马上做调整。隔天再听，或者换不同设备（耳机、车载音响等）听，用新鲜的耳朵发现盲区。

### 2.3 常见混音错误

- 一上来就 Solo 单个乐器开始处理 —— 混音是关系工作，Solo 单轨会让你脱离整体
- 在构思不清晰时就开始拧旋钮 —— 方向不对，参数调得越精细浪费的时间越多
- 不敢做大幅度调整 —— 当你觉得好像这个乐器有点抢时，先拉掉 6dB 再做判断，通常拉 6dB 以上才能听出真正的区别

## 三、分组处理

### 3.1 为什么需要分组

- 同类乐器在混音中往往承担相似的角色，可以在分组上统一处理
- 分组后可以通过一个推子控制多个同类型乐器的音量（如所有打击乐、所有 Pad）
- 分组上可以做统一的空间效果（Reverb、Delay），让同类乐器听起来属于同一空间

### 3.2 分组方法与层级逻辑

#### 分组层级（从下到上）
```
最终输出（Stereo Out）
   |
   +- 总线处理（Master Bus / 母线效果器）
   |
   +- 效果发送组（FX Sends）
   |    +- Reverb Return（板混响/厅混响/房间混响等）
   |    +- Delay Return（Ping-Pong Delay / Slap Delay 等）
   |
   +- 乐器分组（Instrument Groups）
   |    +- 鼓组（Drums）：底鼓、军鼓、Hi-Hat、Tom、Overheads
   |    +- 贝斯组（Bass）
   |    +- 和弦乐器组（Chords）：钢琴、吉他、弦乐、Pad
   |    +- 旋律乐器组（Melody）：Lead、Pluck
   |    +- 人声组（Vocal）
   |
   +- 单轨（Individual Tracks）
```

#### 分组顺序的影响：石头到沙子到水

用一个比喻来理解分组处理的顺序：

- **石头（鼓 + 贝斯）**：节奏的骨架，低频占主导。先处理它们，把节奏和低频定下来。鼓和贝斯的频率关系要优先解决（底鼓 vs 贝斯在 60~80Hz 的避让）。
- **沙子（主要旋律/和弦乐器）**：填充石头之间的空间。当鼓和贝斯确定后，再安排钢琴、吉他等中频乐器的位置，看看它们和鼓贝斯之间的频率关系是否和谐。
- **水（Pad、环境音效、氛围）**：填充剩余的空间。Pad 通常占据较宽的频段，当频率空间已被石头和沙子占据后，Pad 做高通滤波或侧链压缩就可以自然融合而不抢戏。

这个顺序不是死的，但提供了一个优先级思维：**先把节奏和低频定好，再考虑其他**。

#### 具体分组建议

- **底鼓（Kick）和贝斯（Bass）**：通常需要同组处理，因为它们的低频部分容易冲突。Kick 的基频通常在 60~80Hz 区间，贝斯可以稍低或与之错开（如 Kick 在 80Hz 打点、Bass 在 50~60Hz 做铺底）。
- **打击乐组（Drums Bus）**：套鼓（Kick + Snare + Hi-Hat + Cymbals + Toms）放到一个组上做统一的压缩和饱和，让鼓听起来是一个整体。
- **和弦乐器组（Chord Bus）**：钢琴/吉他/Pad 等提供和声背景的乐器，统一在 Bus 上做 EQ 切削（削中低频 200~400Hz 让它们不糊）、统一空间混响。

---

### 3.3 子分组（Sub-Group）

当你的混音工程有大量同类轨道时，可以在分组内再加一层子分组：
```
鼓组 -> 底鼓+贝斯子分组 / 军鼓+镲片子分组
```
这样可以在 Drum Bus 的统一处理基础上，对低频和中高频分别控制。

## 四、EQ（均衡器）

### 4.1 EQ 的本质

EQ 不是用来让一个乐器好听的，而是用来**处理乐器之间的频率关系**——把每个乐器安排到自己的频段上，让它们在频谱上不打架。

EQ 只有两个操作：**提升（Boost）和衰减（Cut）**。但衰减比提升重要得多。

### 4.2 常见 EQ 类型

| 类型 | 特点 | 常用场景 |
|------|------|---------|
| 搁架式（Shelving） | 从某个频率点开始，所有更高/更低频率一起提升或衰减 | 低频切（Low Cut / HPF）、高频空气感提升 |
| 钟形（Bell） | 以某个频率为中心，提升或衰减一个范围 | 精确处理有问题的频点（如鼻音 400Hz、齿音 7~10kHz） |
| 高通滤波（HPF） | 切掉指定频率以下的所有内容 | 去除不需要的低频（人声、吉他、镲片的低频噪音） |
| 低通滤波（LPF） | 切掉指定频率以上的所有内容 | 去除过亮的频率，或模拟电话音效 |

### 4.3 EQ 操作实战：底鼓（Kick）

- 底鼓的穿透力频段（Attack / Click）：**3~5kHz** 提升可以增加打击感的穿透力
- 底鼓的力度频段（Body）：**60~100Hz** 衰减或提升影响底鼓的身材
- 底鼓和贝斯冲突时：将 Kick 的 60~80Hz 保留，Bass 切掉此频段，或者 Bass 保留更低（50~60Hz）让 Kick 的基频清晰

### 4.4 EQ 操作实战：军鼓（Snare）

- 军鼓的共鸣（Body）：**200~400Hz** 附近
- 军鼓的打击感（Crack / Snap）：**5~10kHz** 提升可以增加军鼓的打击冲击感
- 军鼓的饱满度（Weight）：**150~250Hz** 如果有过多共振可衰减

### 4.5 EQ 操作实战：贝斯（Bass）

- 贝斯基频：**40~60Hz**（超低频，需要好的监听系统才能准确听到）
- 贝斯的攻击：**1~2kHz** 提升可以增加贝斯的拨弦清晰度
- 贝斯的指板噪声：**2.5~4kHz** 有过多时可衰减
- 注意：贝斯不要和底鼓在 60~80Hz 打架。通常是底鼓的基频在那里，贝斯切一刀让出空间；或者反过来。

### 4.6 EQ 操作实战：人声（Vocal）

- 低频切除（HPF）：通常从 **40~80Hz** 开始，切掉人声中的低频噪音、气流音
- 鼻音（Nasal）：**300~500Hz** 如果有鼻音过多的问题，衰减 3~6dB（Q 值较宽）
- 温暖感（Warmth）：**150~300Hz** 适度加一点点可以增加人声厚度
- 齿音（Sibilance）：**5~10kHz** 频率范围，太严重时用 De-Esser，而不是 EQ。用 EQ 大量衰减此频段会让人声发闷
- 空气感（Air）：**10~15kHz** 搁架式提升，增加高频空气感，让声音听起来更有质感

### 4.7 HPF（高通滤波）的用法

高通滤波是混音中最常用的操作。几乎每个非低频乐器都可以用 HPF 切掉不必要的低频能量。

- **人声 HPF**：40~80Hz
- **电吉他 HPF**：80~120Hz
- **钢琴 HPF**：60~100Hz（取决于音区）
- **Pad HPF**：100~200Hz
- **Hi-Hat / Cymbal HPF**：200~400Hz
- **Room Mic / Overheads HPF**：100~300Hz

**为什么要做 HPF**：低频能量非常贵。混音中的低频总容量是有限的，如果不把非低频乐器的低频切掉，它们会白白占用宝贵的低频空间，导致底鼓和贝斯在低频区段被污染。

### 4.8 EQ 操作黄金规则

1. **先衰减，后提升**：能用 Cut 解决的就不要 Boost，Cut 是清理空间，Boost 是增加新的问题
2. **衰减更宽的 Q，提升更窄的 Q**：Cut 时 Q 值宽一点（影响范围大），把整个频段清理干净；Boost 时 Q 值窄一点，只精确提升目标频点
3. **Solo 单轨找问题频率，Disable Solo 做判断**：先用 Solo 找到需要 Cut 的频点，然后 Disable Solo 在整体中听这个 Cut 是否合适
4. **不要超过 +/-6dB**：达到这个量时，你可能需要重新考虑乐器本身的选音色问题

## 五、动态处理器（Compressor / Limiter / Gate）

### 5.1 压缩器（Compressor）

#### 压缩器在混音中的作用

1. **控制动态范围**：把乐器中最响的部分压下去，然后整体音量 Makeup Gain 提起来，让弱音也更清晰
2. **增加音乐的挤劲儿**：压缩后的声音更紧凑、更有力度感（Punch）
3. **粘合同一组乐器**：将 Drum Bus 上轻压 2~3dB，能让套鼓听起来是一套而不是散件

#### 四个核心参数

| 参数 | 作用 | 常用设置参考 |
|------|------|-------------|
| Threshold | 达到多大声开始压缩 | 观察 Gain Reduction 表，控制在 2~6dB 之间 |
| Ratio | 超过 Threshold 后压多狠 | 人声 2:1~4:1，鼓组 4:1~8:1，贝斯 3:1~5:1 |
| Attack | 超过 Threshold 后多快开始压 | 慢 Attack（10~30ms）保留打击感；快 Attack（1~5ms）控制瞬态 |
| Release | 低于 Threshold 后多久停止压 | 快 Release（20~50ms）让压缩跟随音乐节奏；慢 Release（100ms+）平滑但可能泵动 |

#### Attack 时间对打击感的影响

- **慢 Attack（10~30ms）**：打击乐器的瞬态先通过再被压缩，保留了打击的啪感（Punch）
- **快 Attack（1~5ms）**：瞬态在爆发前就被压住，声音更平滑但打击感变弱

#### Release 时间对节奏感的影响

- **快 Release（与音乐速度同步）**：压缩器在每拍之间恢复，配合节奏的呼吸感
- **慢 Release**：压缩器持续作用，音量变化更平滑

**Release 时间参考公式**：60,000 / BPM = 每拍毫秒数。如果你的歌 120BPM，每拍 500ms，Release 设在 500ms 或其倍数会让压缩跟随歌曲节奏。

#### 增益衰减表（Gain Reduction Meter）

这是压缩器上最重要的仪表，不用耳朵听就靠它。混音中常见的 GR 范围：
- 单轨轻度压缩：2~4dB
- 组总线压缩：3~6dB
- 母线立体声压缩：1~3dB（非常轻）

#### 常见压缩器类型

| 类型 | 特点 | 代表插件 |
|------|------|---------|
| FET | 反应极快，声音激进 | CLA-76、1176 |
| VCA | 精准、干净、反应较快 | SSL Bus Comp、API 2500 |
| Opto | 反应慢、平滑、温暖 | LA-2A、LA-3A |
| Vari-Mu | 平滑、音乐性强 | Fairchild 670、Manley Variable Mu |

---

### 5.2 限制器（Limiter）

限制器本质上是压缩器 Ratio 无限大（8:1）。它的作用是：**任何超过 Threshold 的信号都被直接拦住，不会更高**。

- **单轨限制**：极少在单轨上用 Limiter，除非目的是限制一个极度过冲的乐器峰值
- **总线限制**：混音最后一步的母带限制，常用 L2、L3、Pro-L2 等，用于将最终峰值控制在 0dB 以下
- **常用设置**：Output Ceiling 设在 -0.5~-1dB（防止 Intersample Peaks），Threshold 拉到 GR 不超过 3dB

### 5.3 门限器（Gate）

门限器用来关掉音量低于某个 Threshold 的信号。常用场景：

- **军鼓底噪控制**：军鼓轨道上设 Gate，在不打鼓时关掉周边镲片漏音
- **底鼓串音隔离**：底鼓麦克风会收到 Hi-Hat 的串音，用 Gate 只在 Kick 打时打开
- **环境噪音消除**：在有空调/电脑风扇背景噪音的环境中录制的轨道

参数：
- Threshold：低于此音量就关门
- Attack：信号超过 Threshold 后多快打开门（保持瞬态）
- Release：信号低于 Threshold 后多快关门
- Hold：关门前的保持时间（防止在音符之间的空隙频繁开关）

## 六、饱和类效果器

### 6.1 饱和的本质

模拟时代的设备（磁带机、电子管放大器、变压器）在处理信号时会自然引入谐波失真，这种失真被称为饱和。饱和本质上是对声音信号做**温和的削波**——在录得不够响时，用饱和增加存在感和厚度。好的饱和不只是失真，而是给声音加上自然的奇次/偶次谐波，让人耳觉得这个声音听着更贵了。

### 6.2 常见的饱和类型

| 类型 | 谐波特征 | 声音感觉 | 代表插件 |
|------|---------|---------|---------|
| 磁带饱和（Tape） | 引入偶次谐波为主，同时轻微压缩高频 | 温暖的、复古的、柔和的 | Studer A800、J37、Kramer Tape |
| 电子管饱和（Tube） | 以偶次谐波为主 | 温暖的、厚实的 | Thermionic Culture Vulture、Decapitator |
| 晶体管饱和（Transistor） | 以奇次谐波为主 | 更脏的、更激进的 | Trash、Decapitator |
| 变压器饱和（Transformer） | 同时引入偶次和奇次，低频略微饱和 | 给声音增加重量感 | Neve 88RS Channel Strip |

### 6.3 饱和在混音中的使用场景

- **鼓组 Bus**：在 Drum Bus 上加一点点磁带饱和，让整套鼓听起来更粘、更贵
- **贝斯**：贝斯经过饱和后，中高频出现失真泛音，在手机/笔记本等低频表现差的设备上也能被听到
- **人声**：在副歌人声上叠加饱和（偶次谐波为主），让人声更有存在感，不用推大音量
- **整曲母线**：非常轻微（Drive 控制在 1~2dB 以内）的母线饱和，模拟整首歌经过了一台磁带机
- **电钢琴 / 合成器**：用饱和增加复古感、Lo-Fi 质感

### 6.4 饱和使用原则

- **适量为主**：拿不准时就开 A/B 对比。如果 A/B 时你分不清饱和在不在，那说明量刚好。如果 A/B 时你一听就知道哦这是开了饱和，那可能是太多了。
- **用并行的 New York 压缩思路**：把饱和的轨道 Send 到一个辅助轨，只混合一部分饱和声，保留原始干净信号
- **低频乐器更容易产生可听感**：饱和在贝斯上加比在镲片上更有感知度

## 七、空间类效果器（Reverb / Delay）

### 7.1 混响的本质

混响的物理本质是声音在空间中反射产生的尾音。混响在混音中的作用比其他效果器更偏向感觉：

1. 营造空间感：让听众感觉到声音所处的空间（卧室/录音棚/音乐厅/教堂）
2. 控制前后纵深：混响多 = 乐器在远处；混响少 = 乐器在近处
3. 声音融合：给不同乐器一个相同的房间混响，让它们在同一空间里
4. 辅助情绪表达：大混响 = 戏剧化/梦幻感；小混响 = 直接/亲密感

从操作的角度，混响是一种帮你把元素放到远的位置的工具，混响越大越靠后。

### 7.2 常见混响类型

| 类型 | 特点 | Pre-Delay 参考 | Decay Time 参考 | 使用场景 |
|------|------|---------------|----------------|---------|
| 板混响（Plate Reverb） | 模拟金属板振动，明亮、有光泽 | 10~30ms | 1.5~3s | 人声、军鼓、打击乐 |
| 房间混响（Room Reverb） | 模拟真实房间的声音，自然 | 0~10ms | 0.5~1.5s | 乐器的深度层，做自然的空间感 |
| 厅堂混响（Hall Reverb） | 模拟音乐厅，浑厚、空间感强 | 20~50ms | 2~4s | Pad、弦乐钢琴等想做大空间感的乐器 |
| 弹簧混响（Spring Reverb） | 复古的弹簧声，颤动的尾音 | 较短 | 1~3s | 电吉他、复古音色 |
| 卷积混响（Convolution Reverb） | 基于 IR（脉冲响应）的真实空间采样 | 取决于采样 | 取决于采样 | 需要真实空间感时 |

### 7.3 混响的关键参数

- **混响时间（Decay / RT60）**：声音衰减 60dB 所需的时间。长混响 = 大空间，短混响 = 小空间
- **预延时（Pre-Delay）**：干声发出后到混响尾音开始的时间。Pre-Delay 越大，干声和混响分离得越开，干声听起来靠前；Pre-Delay 越小，干声淹没在混响中
- **干湿比（Dry/Wet Mix）**：干声和混响声的混合比例。在 Send 轨上用 100% Wet，通过发送量控制干湿比（推荐做法）；在 Insert 上用 Dry/Wet Mix 控制
- **扩散（Diffusion）**：混响中反射声的密度。高扩散 = 丝绸般平滑；低扩散 = 更清晰的反射纹理
- **高低频衰减（HF Damping / LF Damping）**：大部分真实空间中高频比低频衰减更快（声音越远，高频越少）。因此在混响中减少一点高频会让混响听起来更真实

### 7.4 Delay（延时）

Delay 的本质是把声音重复一次或多次。与混响不同，Delay 产生可辨认的回声。

#### 常见类型

| 类型 | 特点 | 常用设置 |
|------|------|---------|
| 1/4 音符 Delay（Quarter Note） | 每拍一个回声，和 BPM 同步 | 60,000 / BPM ms |
| 1/8 音符 Delay（Eighth Note） | 每半拍一个回声 | 30,000 / BPM ms |
| Dotted 1/8 Delay（附点八分） | 最有特色的 Delay 节奏 | 45,000 / BPM ms |
| Ping-Pong Delay | 左右声道交替出现的回声 | 每拍或每半拍交替一次 |
| Slap Delay（拍打回声） | 极短（50~120ms）的单个回声 | 模拟早期录音中从磁带头的回放 |
| Tape Delay | 模拟磁带延时器，回声会逐渐变暗 | 重复次数少，高频自然衰减 |

#### Delay 使用原则

- **人声**：常用 Dotted 1/8 或 1/4 音符 Delay，作为背景填充（Send 轨，湿声调低）
- **乐器的节奏感**：Delay 可以让简单旋律变得有空间律动感
- **不要过堆**：Delay 重复次数（Feedback）通常设置在 1~3 次左右，太多会让混音变糊

### 7.5 Reverb + Delay 配合使用

- Reverb 和 Delay 总是在 **Send 轨**上使用（Send = Aux/Return 轨，100% Wet）
- 好处：一个混响 Send 可以发送给多个乐器，减少 CPU 消耗，同时给所有乐器一个共同的空间感
- 常用搭配：一个 Room Reverb（给整体空间感）+ 一个 Plate Reverb（给人声/军鼓）+ 一个 Delay（给人声/旋律乐器）

## 八、母带处理（总线处理 / Master Bus）

### 8.1 母带处理的目标

母带处理不是拯救一个坏混音的手段。如果混音有问题，母带处理解决不了——它只能放大问题。母带处理的目标是：

1. 让整首歌听起来更专业（响度够、频段平衡、动态适中）
2. 让多首歌在专辑中听起来统一
3. 格式转换（如 24-bit -> 16-bit WAV / MP3）

### 8.2 母线（Master Bus）常用的处理器链

母线处理不是必须的，但绝大多数商业混音会做，处理量通常非常轻柔：

```
Master Bus Chain（仅供参考，不是固定公式）：
    EQ（高频+1~2dB 空气感，低频切一点 30Hz 以下的不必要轰鸣）
  -> Compressor（Bus Compressor，Ratio 2:1，轻压 1~3dB，Attack 10~30ms，Release Auto 或按歌曲速度设置）
  -> Limiter（Output Ceiling -1dB，GR 不超过 3dB，用于控制最终电平）
```

### 8.3 母线压缩的常用设置

- **Ratio**：通常 2:1，不超过 4:1
- **Attack**：10ms 或 Auto（慢到足以让瞬态通过）
- **Release**：Auto 或按歌曲 BPM 设置的较快的 Release（跟随音乐律动）
- **Gain Reduction**：1~3dB（太多会压死动态）

母线压缩的触感效果：当鼓打响时，压缩器短暂地压一下，然后在鼓的间隙恢复——这种呼吸感会让整首歌在节奏上更有推动力。

### 8.4 母带限制器

- **Threshold**：往下拉直到 Gain Reduction 表显示 2~3dB
- **Output Ceiling**：通常设在 -0.5 到 -1dB，避免 Intersample Peaks（高于 0dB 的数字信号在 DA/DA 转换时产生的过载失真）
- **限制器的选择**：Pro-L2 是目前最常用的之一，界面直观、声音干净

### 8.5 参考轨（Reference Track）的使用

- 找 1~2 首和你歌风格相同、混音质量满意的商业作品
- 导入 DAW，和你的混音切换对比
- 对比时注意：整体响度、低频量感、人声清晰度、声场宽度
- 不要追求和参考轨一样响，先追求频段平衡和声场布局接近，再通过母带处理提响度

### 8.6 Bounce（导出）格式建议

- **混音导出**（送往母带）：48kHz / 24-bit / WAV
- **母带后导出**：44.1kHz / 16-bit / WAV（CD 标准）或 48kHz / 24-bit（流媒体平台标准）
- **不要做 dither** 直到最后一步从 24bit 降到 16bit

## 九、常见混音问题与排查思路

### 9.1 混音听起来糊、分不开

- 原因：各乐器的频段重叠太多
- 检查顺序：
  1. 所有非低频乐器是否做了 HPF？（人声、吉他、键盘、镲片等）
  2. 在低频区域底鼓和贝斯是否打架？——用 EQ 给一个让出 60~80Hz
  3. Pad 是否占了太多频段？——Pad 做 HPF 到 150~200Hz，减少和声乐器重叠
  4. 中频 200~500Hz 是否淤积？——用 EQ 在此范围做宽 Q 衰减 2~3dB 尝试

### 9.2 混音听起来平、没有起伏

- 原因：动态控制过度，或者段落之间没有音量/配器层次差异
- 检查顺序：
  1. 压缩器上 Gain Reduction 是否超过 6dB？——降 Ratio 或升 Threshold
  2. 编曲本身是否有层次差异？——Verse 和 Chorus 的乐器数量/音量是否拉开差距
  3. 母带限制器是否 GR 超过 3dB？——升 Threshold 给混音呼吸空间

### 9.3 人声被乐器淹没

- 原因：人声的频率/声场/音量三方面都没有得到保护
- 处理方案：
  1. 音量上：人声在 Chorus 中推大 1~2dB
  2. 频率上：用 EQ 在乐器上切掉人声所在的频段（约 2~4kHz）1~3dB
  3. 声场上：把人声放在正中间，两侧乐器 Pan 开，减少与人声的遮挡
  4. 动态上：人声用压缩控制动态（2:1~4:1, 2~4dB GR），让人声始终保持在可见的水平
  5. 侧链压缩：在人声轨道触发时，背景乐器（尤其是吉他/钢琴）被短暂压下去（2~3dB GR，Attack 快 Release 快）

### 9.4 贝斯和底鼓打架

- 这就是混音中低频管理的核心问题。
- 处理方案：
  1. **频率避让**（最常见做法）：Kick 保留 60~80Hz，Bass 切 60~80Hz；或 Bass 保留低频，Kick 提升 3~5kHz 的穿透力峰
  2. **节奏错开**：Kick 打正拍，Bass 走切分/反拍，两者在时间轴上不挤在一起
  3. **侧链压缩**：Bass 被 Kick 触发侧链压缩，每次 Kick 打时 Bass 低 2~4dB（Attack 快 Release 快）
  4. **分组压缩**：把 Kick 和 Bass 放到一个子分组（Sub Group）上统一轻压，让它们更粘合

### 9.5 在普通设备上听起来不好

- 这是几乎所有初学者都会遇到的问题。在好监听上混了，拿到手机/笔记本/车载上一听完全不一样。
- 解决方案：
  1. 在混音过程中多次用不同的参考系统切换听（耳机、蓝牙音箱、笔记本喇叭）
  2. 确保 200~500Hz 频段没有被淤积（混音发闷的主因之一）
  3. 确保 3~5kHz 频段有清晰度（人声穿透力在此）
  4. 在母带阶段输出的 LUFS（响度标准）不要过低（流媒体通常 -14 LUFS 集成响度）
  5. 使用参考轨对比，校准你的主观判断

## 十、实战案例解析

### 10.1 分轨混音流程演示（鼓组）

**前提**：这是一套软件采样鼓（如 EZDrummer、Superior Drummer 等）。如果是真鼓录音，流程会略有不同。

#### 第一步：听素材，定关系
- 先放鼓 Loop，了解每个鼓件的音色特性
- 这套鼓的底鼓听起来偏软，缺少高频穿刺感？军鼓有腔体共鸣过大？Hi-Hat 音量偏小？

#### 第二步：单轨处理

1. **底鼓（Kick）**
   - HPF：30~40Hz（切除不需要的超低频轰鸣）
   - EQ 提升 3~5kHz 2~4dB（增加打击穿透力，click 感）
   - EQ 衰减 300~400Hz 1~3dB（如果有纸板感或空洞感）
   - Compressor：Attack 稍快（10ms），Ratio 4:1，GR 2~4dB（控制底鼓的动态，保持一致性）
   - 或者用饱和替代压缩：增加底鼓的厚度和存在感（尤其是软件采样鼓）

2. **军鼓（Snare）**
   - HPF：100~150Hz（但保留 150~250Hz 的军鼓共鸣）
   - EQ：如军鼓偏薄 -> 150~250Hz 提升 1~3dB；如打击感不足 -> 6~10kHz 提升 2~4dB
   - 如果有腔体共鸣的嗡嗡声 -> 200~400Hz 窄 Q 衰减 3~6dB（用 Solo 找共振频点）
   - Compressor：Attack 稍慢（20~30ms）保留打击瞬态，Ratio 4:1~6:1，GR 3~5dB
   - Reverb（Send 到 Plate Reverb）：发送量适当，让军鼓有一点光泽而不被淹没

3. **Hi-Hat / Overheads**
   - HPF：200~400Hz
   - Compressor（可选，但通常不需要）：如有必要，Attack 快（5ms），Ratio 2:1，GR 1~2dB
   - Pan：根据真实鼓组定位（Overheads L/R 拉开，Hi-Hat 稍偏左）

#### 第三步：鼓组总线（Drum Bus）处理

所有鼓轨道路由到一个 Drum Bus，然后做：

1. **Drum Bus EQ**：整体鼓组的高频可以在 10~15kHz 微微提升（增加空气感），低频 40~60Hz 微微提升（或用饱和替代）
2. **Drum Bus Compressor**：Attack 10~30ms（保留瞬态的冲击感），Ratio 2:1~4:1，GR 2~3dB（让套鼓粘在一起）
3. **饱和**：在 Drum Bus 上挂一个 Tape 类型的饱和插件，Drive 开到输出指示 1~3dB 的轻微饱和
4. **平行压缩**（可选）：Drum Bus Send 到一个辅助轨，辅助轨上做极限压缩（Attack 快，Ratio 8:1+，GR 10~12dB），然后把这个压扁的鼓信号和原鼓信号混合——这能让鼓在保持动态的同时还有一个结实的骨架声

---

### 10.2 分轨混音流程演示（贝斯）

1. **HPF**：30~40Hz（去掉完全听不到的极低频）
2. **处理底鼓和贝斯的冲突**（参考第 9.4 节）。如果用侧链压缩方案：
   - 贝斯轨上插入 Compressor
   - 侧链输入设置为 Kick 轨道
   - Attack 快（1~5ms），Release 快（20~40ms），Ratio 4:1，GR 2~4dB
   - 每次 Kick 打时贝斯被短暂压一下，两者就不打架了
3. **EQ**：如果需要更清晰的拨弦声，提升 1~2kHz 2~4dB
4. **饱和**：在贝斯上挂一个饱和（Tape 或 Tube），一方面给贝斯增加中高频泛音（让它在小喇叭上也能被听到），另一方面给贝斯增加压缩感
5. **Compressor**：Attack 较快（5~10ms），Ratio 3:1~5:1，GR 3~5dB

---

### 10.3 分轨混音流程演示（人声主轨）

**前提**：这是一条已经录制好的主唱轨，无严重环境噪音问题。

1. **修音/剪辑**（前置步骤，不属于混音但混音前要完成）：人声剪辑对齐节奏，音准修正（Melodyne / VariAudio）
2. **HPF**：40~80Hz（取决于人声的音域，男声可以稍低，女声可以稍高）
3. **如果存在明显的齿音（Sibilance）问题**：挂 De-Esser，中心频率通常在 5~10kHz，衰减 3~6dB
4. **Compressor**：
   - 如果是想让人声更稳更在前面：Attack 5~15ms，Ratio 3:1~4:1，GR 3~5dB
   - 如果只是想控制最大峰值而不改变性格：Attack 20~30ms，Ratio 2:1~3:1，GR 2~3dB
   - 如果用了 1176（FET）：通常用 4:1 或 8:1，Attack 开到最快，Release 开到最慢（All In 按钮模式），GR 4~8dB
5. **EQ**：
   - 衰减鼻音频段（300~500Hz，宽 Q 3~6dB 视情况）
   - 高频 10~15kHz 搁架提升 1~3dB（增加空气感）
6. **Reverb（Send 到 Plate Reverb）**：Pre-Delay 20~40ms，发送量以听出空间感但不淹没干声为准
7. **Delay（Send 到 Dotted 1/8 Delay）**：发送量非常低，只作为背景填充

---

### 10.4 关于乐器摆位的参考原则

乐器在立体声场中的摆放位置参考：

- **最前方（最靠前、最干）**：主唱、独奏乐器
- **中间层**：节奏吉他（左右 Pan）、钢琴、弦乐组（左右分开）
- **最远处（混响最大）**：Pad、环境音效、背景和声
- **居中（左右均衡）**：底鼓、贝斯、主唱
- **左右对称**：两把相同的节奏吉他（Pan 一左一右对称摊开）、Overheads（L/R）
- **特殊定位**：打击乐可以分布在各个位置营造立体感

## 附录：混音检查清单

完成混音后，逐项自检：

- [ ] 混音目标明确了吗？（听素材阶段是否有了声音画面预期？）
- [ ] 监听环境设置正确吗？（音箱等边三角形、高频单元齐耳？）
- [ ] 分组结构建立了吗？（Drums Bus / Bass Bus / Chord Bus / FX Sends？）
- [ ] 所有非低频乐器都做了 HPF 吗？
- [ ] 底鼓和贝斯在低频段没有打架吗？
- [ ] 人声是否清晰可辨，没有被背景乐器淹没？
- [ ] 压缩器的 Gain Reduction 是否控制在合理范围？（单轨 2~6dB，母线 1~3dB）
- [ ] EQ 的 Boost/Cut 是否没有超过 +-6dB？（超过的话考虑换音色）
- [ ] 混响和 Delay 是否以 Send 方式使用而不是 Insert？
- [ ] 母带限制器的 Output Ceiling 设在 -0.5 到 -1dB 了吗？
- [ ] 在不同回放设备上试听过了吗？（耳机、音箱、手机、车载）
- [ ] 和参考轨对比过吗？频段平衡和声场布局是否接近？
