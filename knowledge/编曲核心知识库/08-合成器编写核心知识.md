# 合成器编写核心知识

## 1. 基础概念

### 1.1 什么是合成器

合成器与真实乐器（钢琴、弦乐等）最本质的区别在于：**它把创造音色的权利交给了编曲者**。

用一台合成器，你可以：
- 做出一个属于自己的乐器
- 做出跟真实世界中任何乐器都不一样的乐器
- 不但可以设计它的**质感**（音色），还可以设计它的**演奏技法**（动态）

简单来说，合成器就是一个**音色质感的设计工具**。你不仅是在"弹"它，更是在"造"它。

### 1.2 合成器的本质：两个维度

合成器设计音色的核心思路围绕两个维度：

1. **音色质感**：通过 OSC（振荡器）选择基础波形，再经过 Filter（滤波器）和 FX（效果器）做染色处理，形成特定的音色质感。
2. **演奏技法**：通过 ENV（包络）和 LFO（低频振荡器）控制各个参数随时间的变化，让音色产生动态感。

### 1.3 信号流路径

合成器的核心信号路径：

```
OSC（振荡器）→ Filter（滤波器）→ FX（效果器）→ Amp（放大器）
```

控制流路径：

```
ENV（包络）/ LFO（低频振荡器）→ 调制 OSC / Filter / FX / Amp 上的任意参数
```

### 1.4 减法合成

这里重点使用**减法合成（Subtractive Synthesis）**——目前最主流的合成方式。

思路：从包含丰富谐波的波形（如锯齿波、方波）出发，用滤波器"减去"不需要的频率成分。

类比：就像雕塑，从一块大石头（丰富波形）开始，雕刻掉多余的部分（滤波）得到最终形态。

## 2. 音色分类

### 2.1 Lead（主音）

- **定义**：用于演奏主旋律的合成器音色
- **音色特点**：长线条、有存在感、结实、声音不宽（偏单声道感），手按键盘不松开就会持续发声
- **设计要点**：常用锯齿波或方波为基础，添加适量 Unison 增加厚度
- **音区范围**：C3~C6，与人声主旋律在同一频段
- **常见应用**：副歌主旋律、间奏 Solo、电子舞曲主旋律
- **类比**：独奏弦乐或管乐

### 2.2 Pad（铺底）

- **定义**：用于铺底营造氛围的合成器音色
- **音色特点**：宽阔、柔软、有氛围感，适合演奏和弦
- **设计要点**：多个 OSC 叠加 + Unison/Hyper 展宽 + 混响、延迟
- **音区范围**：C2~C5
- **常见应用**：副歌铺底、氛围段落、段落过渡
- **类比**：合奏的弦乐或管乐

### 2.3 Bass（贝斯）

- **定义**：提供低频支撑的合成器音色
- **音色特点**：可以是长线条（类 Lead 但更低），也可以是短促的（类 Pluck 但更低）
- **设计要点**：常用锯齿波或方波、正弦波（Sub），Filter 低通为主
- **音区范围**：C1~C3
- **常见应用**：EDM 根音律动、Trap 808、House 贝斯线条
- **注意**："Bass"描述的是音区位置（低音），不是演奏技法。同一个 Bass 预设可以是长线条也可以是短促的

### 2.4 Pluck（拨弦）

- **定义**：音头强、衰减快的拨弦类音色
- **音色特点**：音头很强、很快衰减，手一直按着也只会短促地响一下，像吉他拨弦
- **设计要点**：ENV 的 A 很短，D 很快，S 很低或为 0
- **音区范围**：C3~C6
- **常见应用**：节奏型铺底、创造律动、也可以写作旋律

### 2.5 Arp（琶音）

- **定义**：自动按设定模式循环演奏和弦内音的琶音器音色
- **特点**：有节奏感、自动演奏、适合做律动层
- **常见应用**：段落中节奏铺底、过渡段律动

### 2.6 FX（效果音色）

- **定义**：不是常规乐音的音效类音色
- **常见类型**：
  - **Riser**：音高或 Filter Cutoff 持续上升，用于段落过渡或情绪推进
  - **Downlifter**：与 Riser 相反，音高或滤波持续下降，用于段落结束
  - **Impact**：短促有力的撞击声，用于段落开头或重拍强调
  - **Sweep**：滤波快速扫过全频段的声音

### 2.7 其他分类

- **Keys**：类键盘乐器音色（电钢、风琴等）
- **Bell**：钟铃类清脆音色
- **Vocal**：人声类音色（非真实人声，是类似人声质感的合成音色）
- **String**：模拟弦乐质感的合成音色

### 2.8 音色分类的理解

需要注意：
- Lead / Pluck 描述的是**演奏技法**
- Bass 描述的是**音区位置**
- Pad 描述的是**声音的宽窄和功能**

同一个合成器预设：
- 改变 ADSR（让声音变短促），Lead 可以变成 Pluck
- 降低八度，Lead 可以变成 Bass
- 加 Unison 和混响，Lead 可以变成 Pad

## 3. 参数详解

### 3.1 OSC（振荡器 / Oscillator）

OSC 是合成器发出原始声音的模块，是整个信号流的起点。

#### 3.1.1 基础波形

**正弦波（Sine Wave）**
- **听感**：圆润、纯净，像笛子
- **谐波**：只有基频，没有任何谐波
- **示例**：按下 A 音，频谱上只有 220Hz

**锯齿波（Sawtooth Wave）**
- **听感**：明亮、刺激，有很强存在感
- **谐波**：包含所有整数倍谐波，从基频到高频逐渐减弱

**三角波（Triangle Wave）**
- **听感**：介于正弦波和锯齿波之间，比正弦波丰富但比锯齿波柔和
- **谐波**：只有奇次谐波，音量比锯齿波小得多

**方波（Square Wave）**
- **听感**：中空、嗡嗡质感，比锯齿波少一些刺激感
- **谐波**：只有奇次谐波，呈现"一大一小"的间隔模式

**脉冲波（Pulse Wave）**
- **听感**：介于方波和锯齿波之间，可通过 PWM 连续变化

#### 3.1.2 WT POS（波表位置 / Wavetable Position）

- **定义**：在波表中选择不同的波形位置
- **作用**：转动 WT POS 可在不同波形之间平滑过渡
- **应用**：用 ENV 或 LFO 调制 WT POS，实现在演奏过程中波形连续变化

#### 3.1.3 Sub 振荡器

- **常用波形**：正弦波或方波

#### 3.1.4 Noise（噪音）

- **类型**：白噪音、粉红噪音
- **特点**：包含所有频率，没有固定音高

#### 3.1.5 Unison（齐奏）

- **原理**：复制当前声音多份，每份之间做微小音高偏移和声像偏移
- **参数**：
  - **Count**：复制份数（2~16 份）
  - **Detune**：音高偏移量（0~100 cent），越大声音越宽但音调越模糊
  - **Pan**：声像分布范围
- **效果**：从单声道变成立体声
- **注意**：需要旋律清晰时 Detune 小一些，需要铺底时 Detune 大一些

#### 3.1.6 PWM（脉冲宽度调制）

- **适用**：脉冲波
- **原理**：周期性改变脉冲占空比
- **效果**：动态变化的音色质感，类似合唱效果器
- **实现**：用 LFO 调制 Pulse Width 参数

#### 3.1.7 OSC Level

- 控制每个振荡器输出的音量大小
- 这只是 OSC 的最大输出音量，最终音量还受 AMP ENV 控制

### 3.2 Filter（滤波器）

Filter 是改变音色频率成分的核心模块，位于 OSC 之后。

#### 3.2.1 滤波器类型

**LPF（低通）**
- 让低于截止频率的声音通过，切掉高频
- 作用：让声音变暗、变柔和
- 典型应用：Wobble Bass、自动关门效果

**HPF（高通）**
- 让高于截止频率的声音通过，切掉低频
- 作用：让声音变薄、变细，弱化低频

**BPF（带通）**
- 让中间一段频率通过，切掉高频和低频

**Notch（陷波）**
- 切掉中间一段频率，保留高频和低频

**组合模式（Morph）**
- 一个旋钮让滤波器从 LPF→BPF→HPF 渐变
- 最左边 LPF，中间 BPF，最右边 HPF

#### 3.2.2 Cutoff（截止频率）

- **定义**：滤波器开始工作的频率点
- **单位**：Hz，范围 20Hz~20kHz
- **典型设置**：
  - 温暖 Pad：LPF Cutoff ≈ 2kHz~5kHz
  - 明亮 Lead：LPF Cutoff ≈ 8kHz~全开
  - 暗色 Bass：LPF Cutoff ≈ 200Hz~500Hz
  - HPF 电话声：Cutoff ≈ 300Hz~500Hz

#### 3.2.3 Resonance（共振 / Res）

- **定义**：截止频率点处的音量增益
- **效果**：Res 增大 → 截止频率处形成峰 → "共鸣"音色
- Res 很大时 → 类似啸叫或口哨效果
- **典型应用**：Acid Bass（Res 开大 + LPF Cutoff 用 ENV/LFO 调制）
- **注意**：Res 很大会导致声音发"尖"

#### 3.2.4 Filter Slope

- **定义**：滤波器对截止频率之外声音的衰减速率
- **常见值**：12dB/oct（温和过渡）、24dB/oct（锐利切割）

#### 3.2.5 Drive（驱动）

- **定义**：输入给滤波器的信号增益
- **效果**：Drive 调高 = 音量放大 → 产生失真

#### 3.2.6 Key Track（键盘跟踪）

- **作用**：让 Cutoff 跟随演奏音高自动变化
- 弹高音时 Cutoff 自动升高，弹低音时自动降低
- 让音色在不同音区保持一致的明暗感

#### 3.2.7 Filter ENV

- 用 ENV 调制 Cutoff
- 每次弹奏时 Cutoff 从低到高扫描一次（自动哇音效果）

#### 3.2.8 Pan

- 让滤波器对不同声道起到不同程度的过滤
- 默认 50% 对所有声道一样；左右旋转分别对左/右声道过滤更强

#### 3.2.9 Mix（干湿比）

- **100%**：所有声音都经过滤波（默认）
- **0%**：滤波不起作用
- 想保留部分原始质感时，Mix 调到 50%~80%

### 3.3 FX（效果器面板）

FX 面板是放在合成器内部的效果链，位于 Filter 之后、Amp 之前。效果器是**串联连接**的。

#### 3.3.1 Hyper / Dimension（声场扩展）

**Hyper**
- Unison 展宽逻辑：复制声音 + 音高偏移 + 分散声像
- 参数：Detune（偏移量）、Rate（振荡速度）、Mix

**Dimension**
- 另一种展宽方式，效果比 Unison 更自然
- 参数：Size（展宽程度）、Mix
- 区别：Unison 通过音高差制造宽度，Dimension 通过时间差制造宽度

#### 3.3.2 Distortion（失真）

- **作用**：通过谐波饱和让声音变结实、有力量
- **参数**：Type（类型）、Drive（失真量）、EQ（可选 Pre/Post）、Mix
- **典型应用**：Bass 加少量失真让低频更扎实；Lead 加失真增加穿透力

#### 3.3.3 Flanger / Phaser / Chorus

- **Flanger**：梳状滤波效果，声音旋转移动
- **Phaser**：比 Flanger 更柔和，相位变化
- **Chorus**：让单一音色听起来像多个音色同时演奏
- **参数**：Rate、Depth、Feedback、Mix

#### 3.3.4 Reverb（混响）

- **参数**：Decay（0.1s~10s+）、Pre-delay、Size（Room/Hall/Plate）、Damping、Mix
- **典型设置**：
  - Pad：Decay 3s~5s
  - Lead：Decay 0.5s~1.5s
  - Pluck：适量混响，不要淹没音头

#### 3.3.5 Delay（延迟）

- **参数**：Time（BPM 同步或 ms）、Feedback、Filter、Mix
- **同步值**：1/4、1/8、1/16、1/4T 等
- **典型**：Lead 用 1/4 延迟少量 Mix；Pluck 用 1/8 或 1/16 延迟

#### 3.3.6 Compressor

- **参数**：Threshold（-60dB~0dB）、Ratio（1:1~20:1）、Attack（0.1ms~50ms）、Release（10ms~500ms）、Gain

#### 3.3.7 EQ

- 精细调整音色频率平衡
- 用于合成器内部做最后的频率塑形

#### 3.3.8 效果器顺序调整

- 所有 FX 模块可拖拽改变顺序
- 先失真再混响 vs 先混响再失真产生完全不同结果
- 音色设计阶段应更自由，不拘泥混音规则

### 3.4 ENV（包络 / Envelope）

ENV 是控制参数随时间变化的模块，赋予音色"演奏技法"。

#### 3.4.1 ADSR 四阶段

**A - Attack（起音，0ms~10s+）**
- 短（0~5ms）：音头冲、直接（Pluck、打击音色）
- 中（10~50ms）：自然起音（Lead、Bass）
- 长（500ms+）：声音慢慢出来（Pad）

**D - Decay（衰减，0ms~10s+）**
- 短：很快从峰值掉到保持值
- 长：缓慢下降

**S - Sustain（保持电平）**
- Amp ENV 单位 dB，调制 ENV 单位百分比
- S=100% 按住保持最大，S=0% 按住也会衰减到无声

**R - Release（释放，0ms~10s+）**
- 短：松手马上停止
- 长：松手后慢慢消失

#### 3.4.2 不同音色的 ADSR 参考

| 类型 | Attack | Decay | Sustain | Release |
|------|--------|-------|---------|---------|
| Pluck | 0~5ms | 50~200ms | 0%~20% | 50~200ms |
| Lead | 5~20ms | 200~500ms | 70%~100% | 200~500ms |
| Pad | 200ms~2s | 500ms~2s | 70%~100% | 1s~3s+ |
| Bass | 5~30ms | 100~500ms | 50%~100% | 100~500ms |

#### 3.4.3 ENV 典型应用

**Amp ENV（ENV 1）**：默认控制总音量，决定音色基本形态
**Filter ENV（ENV 2/3 → Cutoff）**：每次弹奏滤波扫描，"哇"效果
**FX ENV（ENV 3 → FX Mix）**：每次弹奏带出混响尾音

#### 3.4.4 多 ENV 协同

Serum 有 3 个 ENV：ENV 1 控制 Amp，ENV 2 调制 Filter Cutoff，ENV 3 调制 Reverb Mix，只用三个 ENV 就能创造丰富动态。

### 3.5 LFO（低频振荡器）

LFO 是产生周期性调制的模块。

#### 3.5.1 LFO vs ENV

| | ENV | LFO |
|--|-----|-----|
| 触发 | 每次触发音符执行一次 | 以固定速率循环运行 |
| 类比 | 一次性动作 | 持续马达 |
| 适用 | 音头、音尾等一次性动态 | Tremolo、Vibrato、Wobble 等循环效果 |

#### 3.5.2 LFO 参数

**Rate（速率）**
- BPM 同步：1/64~16 小节
- 自由频率：0.01Hz~100Hz+
- 典型：Tremolo 1/16~1/8、Wobble 1/8~1/4、慢速声像 1/2~2 小节

**Shape（波形）**
- 正弦波：平滑变化（Vibrato/Tremolo）
- 锯齿波：上升/下降后回弹（Riser）
- 方波：开关式变化（Trance Gate）
- 随机波 S&H：随机跳跃（电子特效）
- Serum 中可自定义绘制

#### 3.5.3 LFO 典型应用

- **Tremolo**：LFO → Amp，Rate 1/16~1/8，正弦波，音量周期性变化
- **Vibrato**：LFO → Pitch，约 5Hz，正弦波，Depth 几 cent~20 cent
- **Wobble Bass**：LFO → Filter Cutoff，Rate 1/8~1/4，锯齿波，Dubstep/EDM
- **Autopan**：LFO → Pan，正弦波，声音左右周期性移动
- **Filter Sweep**：LFO → Cutoff，慢速 1~2 小节，音色周期性变亮变暗

#### 3.5.4 LFO 扩展

Serum 默认 4 个 LFO，最多扩展到 8 个，独立调制不同参数，波形可自由绘制。

### 3.6 Global（全局控制）

#### 3.6.1 Portamento（滑音）

- 范围 0ms~10s+
- 短（20~100ms）：轻微滑动，旋律连贯
- 长（500ms+）：明显滑音，科幻感

#### 3.6.2 Poly / Mono / Legato

- **Poly**：同时演奏多个音符（和弦）
- **Mono**：同一时间只响一个音符（Lead、Bass Solo）
- **Legato**：后音与前音重叠时不触发新 ENV，更连贯

#### 3.6.3 Bend Range

- 弯音轮范围，单位半音
- 常见：±2（默认）、±12（一个八度）
- Lead 和 Bass 常用 ±12

## 4. 氛围设计

### 4.1 什么是氛围设计

氛围设计是指在编曲中，利用合成器、采样、效果器等手段，营造出特定情绪和环境感的过程。

### 4.2 "灯光师"类比

氛围设计可以类比演出灯光师的思路：

- **主光（Key Light）**：最主要的氛围层音色，通常是 Pad 或宽厚的合成器
- **顶光（Top Light）**：高音区的点缀性音色，如 Bell、干净的 Pluck
- **背光（Back Light）**：低频氛围层，如 Sub Bass 或低频 Pad

### 4.3 设计流程

1. **确定感觉**：先明确这段想表达的情绪（温暖/冷冽/紧张/放松）
2. **选择音色类型**：根据情绪选择 Pad / FX / Arp 等
3. **设计参数**：
   - OSC 选波形（温暖→三角波、刺激→锯齿波）
   - Filter 设定 Cutoff（温暖→Low Cutoff ~2kHz）
   - ENV 设定形态（持续还是短促）
   - FX 添加空间感（混响 Decay 3~5s）
4. **与其他乐器的关系**：Pad 不要占用钢琴和中频乐器的频段

### 4.4 常用氛围设计手法

- **Filter Sweep**：LFO 缓慢调制 Cutoff，音色周期性亮暗变化
- **Riser 铺垫**：白色噪音 + Filter 从高到低打开，配合 Reverb 和 Delay
- **分层叠加**：多个 Pad 在不同音区叠加，每个 Pad 占据不同频段

## 5. 音源使用

### 5.1 Serum

**特点**：界面直观、图形化显示波形和调制、适合学习合成器

**界面关键区域**：
- 顶部 OSC A / OSC B：波形选择与波表位置
- 中间 Sub / Noise：辅助发声
- 右侧 Filter：滤波器类型和参数
- 下方 FX：效果器链
- 底部 ENV 1~3 / LFO 1~8：调制模块

**基本操作要点**：
1. OSC A 选择基础波形
2. Filter 选择类型，设置 Cutoff 和 Res
3. ENV 1 调整音色基本形态（ADSR）
4. FX 面板添加混响或延迟
5. LFO 调制关键参数创造动态

## 6. 实战案例

### 6.1 设计一个温暖的 Pad

1. OSC A：选三角波（Triangle），WT POS 打到偏正弦波位置
2. OSC B：选正弦波，比 OSC A 低八度，Level 减半
3. Filter：LPF，Cutoff ≈ 2kHz，Res ≈ 20%
4. ENV 1：A=500ms, D=1s, S=80%, R=2s
5. FX：Reverb Decay=3s, Mix=40%；Delay 1/4, Mix=20%
6. Unison：Count=4, Detune=15 cent

### 6.2 设计一个明亮的 Lead

1. OSC A：锯齿波（Sawtooth）
2. Sub：正弦波低八度，Level 30%
3. Filter：LPF，Cutoff ≈ 8kHz，Res ≈ 10%
4. ENV 1：A=10ms, D=300ms, S=90%, R=400ms
5. FX：Distortion 少量，Reverb Decay=1s, Mix=25%
6. Unison：Count=2, Detune=8 cent（保持旋律清晰）
7. Mode：Mono，Portamento=30ms

### 6.3 设计一个 Wobble Bass

1. OSC A：锯齿波
2. Sub：正弦波低一个八度，Level 50%
3. Filter：LPF，Cutoff 用 LFO 调制
4. LFO 1：锯齿波形，Rate=1/8，调制 Filter Cutoff 量 60%
5. ENV 1：A=5ms, D=200ms, S=80%, R=200ms
6. FX：Distortion 适量，增加低频力量感
