# AutoTune修音后声音反而变怪了怎么回事？

**症状：** 加了AutoTune之后，声音不但没有变好听，反而变得奇怪不对劲。

**原因：** 调式设置与歌曲原调不一致，把C调的歌设置成了D调或其他调，修音反而修偏了。

**解决：** 方法一：使用AutoKey插件 → 插入伴奏轨道 → 让它自动识别歌曲调式 → 点击对应调式（如C2）自动同步到AutoTune。方法二：使用在线测调网站（如Evanion）→ 拖入伴奏音频 → 获取调式信息 → 在AutoTune上选择对应的大调(Major)或小调(Minor)及字母调号。

**常见误区：** 没有先识别调式就随意选择；不知道大调小调的区别（Major是CDEFJAB，Minor是小调）。

**关键词：** #插件
