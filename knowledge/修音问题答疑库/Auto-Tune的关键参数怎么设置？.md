# Auto-Tune的关键参数怎么设置？

**核心要点：** Auto-Tune的关键参数包括Input Type、Retune Speed、Humanize等，设置需根据演唱风格和需求调整。

**详细解答：** Auto-Tune的关键参数设置：

### 1. Input Type (输入类型/音域选择)
- **作用**：告诉Auto-Tune处理的人声音域范围
- **选项**：Soprano（女高音）、Alto/Tenor（女中音/男高音）、Low Male（男低音）、Instrument（乐器）
- **设置原则**：选择最接近演唱者实际音域的选项

### 2. Retune Speed (校正速度)
- **作用**：控制音高校正的速度
- **范围**：0（最慢）到100（最快）
- **常用设置**：
  - **自然效果**：20-40，保留一些人声自然波动
  - **明显效果**：80-100，创造经典的Auto-Tune效果
  - **现场演出**：40-60，平衡自然度和校正效果

### 3. Humanize (人性化)
- **作用**：保留一些人声的自然波动，避免过于机械
- **设置原则**：需要自然感时调高，需要明显效果时调低

### 4. Key/Scale (调性/音阶)
- **作用**：设置歌曲的调性和音阶
- **重要性**：必须正确设置，否则会校正到错误的音高
- **设置方法**：根据歌曲的实际调性和音阶设置

### 5. Correction Mode (校正模式)
- **作用**：选择校正的强度模式
- **选项**：Auto（自动）、Graph（图形）、Manual（手动）
- **推荐**：初学者用Auto，有经验后用Graph获得更多控制

**实用技巧：**
1. **先设Input Type**，确保音域匹配
2. **再设Key/Scale**，确保调性正确
3. **最后调Retune Speed**，根据需求调整效果强度
4. **实时监听调整**，找到最适合的设置

**应用场景：** 1. 使用Auto-Tune时 2. 教学参数设置时 3. 不同风格制作时
