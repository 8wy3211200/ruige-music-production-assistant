# Melodyne 手动修音的核心思路是什么

**症状：** 用户在混音阶段想把音一个一个修准，但不知道为什么有经验的制作人更推荐 Melodyne，而不是继续只靠自动修音。

**原因：** 混音时目标更细致，AutoTune 更适合实时场景，而手动修音更容易针对具体音符做精细判断。

**解决：** 在 Studio One 里直接用 Melodyne 编辑音频，先确认识别模式是 `melodic`，再以音高工具和节奏工具为主，一个音一个音往调内拉；修的时候看参考轨，但仍然坚持“三分看七分听”。

**常见误区：** 以为看着音块拉到格子上就一定对，不需要再听。

**关键词：** #修音, #StudioOne, #人声
