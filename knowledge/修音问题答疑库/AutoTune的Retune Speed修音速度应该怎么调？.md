# AutoTune的Retune Speed修音速度应该怎么调？

**症状：** 不知道Retune Speed该调多少，数值设置完全凭感觉。

**原因：** 不了解这个参数对修音效果的具体影响机制。

**解决：** 数值越低（如5-10）→ 修音越慢越柔和 → 效果自然 → 适合正常修音。数值越高（如50-100）→ 修音越快越激进 → 电音效果越明显 → 适合风格化处理如嘻哈说唱。一般人声修音建议设置在20-40之间。

**常见误区：** 调太低修不准，调太高又变成电音效果不知道该选哪个。应该根据用途选择：正常修音用低值，想要电音风格用高值。
