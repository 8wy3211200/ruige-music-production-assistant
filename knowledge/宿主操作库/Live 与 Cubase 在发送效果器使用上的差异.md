# Live 与 Cubase 在发送效果器使用上的差异

**症状：** 不确定 Live 的发送式效果处理方式是否正确，与其他 DAW 逻辑不同

**原因：** Live 采用"轨道分组+效果链"的直观方式，而 Cubase 等传统 DAW 采用 return 总线的传统调音台逻辑

**解决：** 理解两种方式的本质差异：
- Live 方式：各轨道独立挂载效果→直观但占资源
- Cubase 方式：10个轨道共用同一轨混响→节省资源但逻辑传统

两种方式各有优劣，可根据项目需求选择。Live 的 Audio Effect Rack 可以实现效果链的批量化管理

**常见误区：** 认为必须使用 return 轨道才能实现发送效果，或者认为 Live 的方式"不正规"

**关键词：** #混响, #插件
