# Ableton Live Clip 与 Warp 实战

## 核心概念
- Live 的基本材料是 Clip，Audio Clip 通过 Warp 改变速度和时间位置。
- Warp 是非破坏式时间处理，适合把采样、loop 和录音对齐工程速度。
- Clip View 是设置 start/end、loop、warp、gain、pitch、envelope 的核心界面。
- Warp 模式选择会影响音质和节奏感，不能一律默认。
- 采样编曲先校正 timing，再做切片、变调和效果。

## 使用场景
- 导入鼓 loop、对齐人声、重构采样、做转场、从素材库快速搭节奏。
- 短鼓采样通常用 Beats，旋律和人声要测试 Tones、Texture、Complex。
- 需要保持原音质时，确认 tempo 相同且 Warp/Fade 设置合理。
- 需要创意变形时，故意使用极端 Warp 和 clip envelope。

## 核心参数
- 需要统一 groove 时，把 Warp Markers 和 Groove Pool 配合。
- Warp Markers：锁定关键拍点和瞬态。
- Warp Mode：按素材类型选择 Beats、Tones、Texture、Re-Pitch、Complex。
- Clip Gain：先做素材级增益，不要全靠轨道音量。
- Transpose/Detune：做调性适配和采样重构。
- Loop Region：控制循环范围和段落长度。

## 实战案例
- Clip Envelope：在 clip 内做非破坏调制。
- 人声采样：先标第一拍和句尾关键点，再少量拉伸，不要每个字都打点。
- 鼓 loop：用 Beats 保留瞬态，再 Groove Pool 轻微 swing。
- 氛围采样：Texture 模式拉长，clip envelope 控制滤波和音量。

## 补充提醒
- 转场：复制 clip，末尾用 transpose 上行和 reverb send 自动化。
