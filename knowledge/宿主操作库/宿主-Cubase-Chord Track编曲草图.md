# Cubase Chord Track 编曲草图

## 核心概念
- Chord Track 用来在工程里放置和弦和调性事件，是编曲草图的和声层。
- 它能辅助 MIDI、VariAudio 和跟随和弦的轨道进行和声适配。
- Scale Lane 可显示调式信息，帮助旋律和低音保持一致。
- Chord Track 不替代耳朵，但能让项目结构和和声关系清楚。
- 适合先搭和声框架，再分配钢琴、贝斯、弦乐和旋律。

## 使用场景
- 流行歌 demo、影视主题变体、快速改调、检查旋律是否跑调。
- 需要改副歌和弦时，先改 Chord Track，再同步检查跟随轨道。
- 需要从 MIDI part 搜索和弦时，用 Cubase 的 chord search 工作流。
- 需要 VariAudio 修人声时，可用 Chord Track 数据作为 scale。

## 核心参数
- 需要给队友看结构时，用 chord events 标出段落和声。
- Chord Events：每个位置的和弦标记。
- Scale Events：调式和音阶标记。
- Audition Track：试听 chord event 的轨道。
- Follow Chord Track：让 MIDI 或音频按和弦逻辑跟随。
- Chord Pads：辅助试和弦和 voicing。

## 实战案例
- VariAudio Scale：把和弦轨用于音高校正参考。
- demo 起步：先写 8 小节 Chord Track，再用钢琴和贝斯填充。
- 副歌替换：只改最后两个 chord events，试听旋律是否更有拉力。
- 人声修音：用 Chord Track 提供 scale，避免修到不属于和声的音。

## 补充提醒
- 弦乐铺底：按 chord events 生成长音，再手工调 voicing。
