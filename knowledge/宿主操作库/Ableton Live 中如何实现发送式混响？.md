# Ableton Live 中如何实现发送式混响？

**症状：** 不知道在 Live 中如何做发送式混响，不确定该用哪种方式

**原因：** Live 的 return/反送轨道有局限性，使用后无法做总编组把 return 也包在里面进行统一母带处理

**解决：** 不用 return 轨道，采用干湿分组方式实现发送效果：
1. 将干声比（Dry）拉到 100%
2. 在该轨道上右键选择"分组"
3. 下面链设为干声，上面链挂混响效果器
4. 这样就等于实现了发送式混响

**常见误区：** 直接使用 return 轨道做发送式混响，会导致无法统一管理所有轨道进行母带处理。Live 中 return 轨道是独立的，无法编入总 group

**关键词：** #混响, #插件
