# Studio One 高频快捷键

## 这是什么
- 这页不是快捷键总表，而是“先背这些，日常效率立刻上来”的高频子集。
- 用户最应该先记的是四类：文件与检索、浏览器切页、编辑与量化、导出与定位。
- 如果你把所有快捷键一口气都背，最后通常一个都用不稳。

## 怎么操作
### 1. 文件与检索
- `Ctrl + N`：新建 Session
- `Ctrl + O`：打开
- `Ctrl + S`：保存
- `Ctrl + F`：搜索
- `Ctrl + K`：查找命令

### 2. 浏览器切页
- `F6`：Show Instruments
- `F7`：Show Effects
- `F8`：Show Loops
- `F9`：Show Files
- `F10`：Show Pool

### 3. 编辑与节奏
- `Alt + X`：Split at Cursor
- `Alt + D`：Duplicate and Insert
- `Q`：Quantize
- `Alt + Q`：Quantize 50%
- `Shift + Q`：Restore Timing
- `N`：Toggle Snap
- `Z`：Toggle Zoom
- `Shift + Z`：Store Zoom State
- `Alt + W`：Undo Zoom

### 4. 轨道控制
- `T`：Add Tracks
- `R`：Arm
- `U`：Monitor
- `M`：Mute
- `S`：Solo
- `A`：Show Envelopes

### 5. 导出与定位
- `Ctrl + E`：Export Mixdown
- `Ctrl + Shift + E`：Export Stems
- `L`：Locate Selection
- `Shift + P`：Loop Selection
- `Shift + N`：Goto Next Marker
- `Shift + B`：Goto Previous Marker
- `NumPad1`：Goto Loop Start
- `NumPad2`：Goto Loop End

## 常见误区与排查
- 误区：一上来就背整张表。
  - 正解：先把这页的 20 来个高频命令用熟，再按自己的工作流补。
- 误区：只背按键，不知道它依赖什么上下文。
  - 正解：例如 `Q` 在 MIDI、音频、事件编辑里的对象不同，但本质都是按当前量化面板工作。
- 误区：记不住命令就硬猜。
  - 正解：先用 `Ctrl + K` 打开 `Find Command`，很多命令名字一搜就能定位。
- 误区：觉得默认键位不顺手，只能忍。
  - 正解：官方快捷键表明确写了可以去 `Options / General / Keyboard Shortcuts` 里改，也能套别的 DAW 模板。
- 症状：一部分命令和别人教程不一样。
  - 先确认对方是不是 macOS。
  - 再确认是不是自定义了 `Keyboard Shortcuts` 模板。

## 补充提醒
- 萃取边界：不搬全表，不收冷门命令，只保留最常用、最容易直接提升效率的一批。
