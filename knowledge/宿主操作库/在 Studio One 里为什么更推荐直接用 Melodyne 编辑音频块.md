# 在 Studio One 里为什么更推荐直接用 Melodyne 编辑音频块

**症状：** 用户知道可以把 Melodyne 当普通插件插进去，但流程比较绕，不清楚为什么有经验的制作人更推荐另一种方式。

**原因：** 插件插入模式需要先 `transfer` 一遍音频，录进去多少才能修多少，而且后续编辑和原音频块脱节；Studio One 支持 ARA，可直接把 Melodyne 嵌在音频块上。

**解决：** 在 Studio One 里优先用“右键音频块/双击音频块进入 Melodyne 编辑”的方式工作，直接在原音频上修，效率更高。

**常见误区：** 以为 Melodyne 一定要走“插入插件 + transfer”这条路。

**关键词：** #修音, #StudioOne, #工作流
