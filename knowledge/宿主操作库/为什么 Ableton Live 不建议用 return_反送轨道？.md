# 为什么 Ableton Live 不建议用 return/反送轨道？

**症状：** 想用 Live 的 return 轨道做发送式效果

**原因：** Live 的 return 轨道是独立于主轨道的，使用后只能在 main 总输出上做母带，无法把所有轨道（包括 return）统一编组管理

**解决：** - 如果需要统一管理所有发送效果：使用 Audio Effect Rack（音频效果机架）在各轨道上直接挂载效果器
- 如果想给所有轨道共用同一混响：用干湿分组的方式或复制同一效果链
- 各轨道独立挂载效果虽然占资源，但更直观灵活

**常见误区：** 认为 return 轨道是实现发送效果的唯一方式，忽视了 Live 中轨道分组的灵活性

**关键词：** #混响, #插件
