# 人声double和和声录制技巧

**症状：** 副歌部分做了加倍生成效果感觉不够好，不知道如何处理。

**原因：** 没有在录音阶段直接录制double轨，后期生成的效果不如原始录音自然。

**解决：** 1. **录音时直接录两轨double**：这是原创歌曲制作中非常常见的手法
2. **放在左右声道**：两轨double放在左右两侧增加宽度和厚度
3. **翻唱也可借鉴**：既然原唱用了这种手法，翻唱时同样可以这样处理
4. **后期可再画automation**：录过double之后再画自动化曲线都没问题

**常见误区：** 过度依赖后期效果器生成double，忽略了录音阶段直接录制的重要性；automation不是用来解决人声厚度的核心手段。

**关键词：** #和声, #立体声, #宽度, #人声
