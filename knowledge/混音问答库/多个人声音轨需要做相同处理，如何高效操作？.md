# 多个人声音轨需要做相同处理，如何高效操作？

**症状：** 多个和声音轨需要统一做低切、EQ等相同处理时，逐个添加插件效率低下。

**原因：** 如果每个音轨单独添加相同处理，当和声音轨数量多时（如8个、10个），操作繁琐且不利于后续统一调整。

**解决：** 利用Bus（母线）编组处理：
1. 按住Shift键选中所有需要统一处理的音轨
2. 右键点击 → "根据所选通道添加Bus"
3. 在新建的Bus上添加统一处理插件（如EQ低切）
4. 此后调整Bus上的参数，所有音轨同步变化

**常见误区：** 不必纠结于只有两个音轨时是否值得这样做，实际上3个以上音轨就有明显效率提升。另外Bus层级可以嵌套——先分组处理，再对多组建立总Bus做二次处理。

**关键词：** #插件
