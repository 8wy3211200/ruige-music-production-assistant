# 替换鼓组局部元素时，要尽量保住原 loop 的整体感

**症状：** 想替换鼓 loop 里的某些成分时，容易一换就散，失去原本完整的鼓 loop 感。

**原因：** 示范中，可以把原鼓 loop 当成类似真实鼓里的 overhead 来处理，保留它的整体性，再替换个别核心部件。

**解决：** 替换局部元素时，可以先把原 loop 的低频切掉，再配合侧链等手段，让新换进去的 kick/snare 接管主干，同时保留 loop 原有的整体氛围和粘合感。

**常见误区：** 以为替换鼓元素就是把原 loop 全部推翻重来。

**关键词：** #侧链, #分轨, #混音, #工作流
