# 麻将声学（Audio春虫虫）的HRTF预设如何使用？

**症状：** 拿到了麻将声学的HRTF扫描结果文件，但不知道如何在实际工程中使用；或者用不同品牌EQ时找不到对应预设。

**原因：** 麻将声用户成的HRTF校正包通常包含特定宿主和特定EQ插件的预设，如果用户使用的软件或插件不匹配，预设无法直接加载。

**解决：** 步骤如下：
1. 用水月雨或对应耳机完成3D扫描，获取HRTF文件
2. 效果链顺序：Vertoso（空间化）→ 专用EQ（频响补偿）
3. 如使用Ableton Live + FabFilter EQ：直接加载对应预设
4. 如使用其他宿主/其他品牌EQ：需要手动匹配参数（"照着它这个去调一个"）
5. Windows系统建议配合Flex ASIO + Element机架使用

**常见误区：** 直接套用预设而不理解效果链逻辑；使用不兼容宿主时硬套预设参数导致效果不佳；忽略了EQ补偿这一环。

**关键词：** #混响, #EQ, #插件
