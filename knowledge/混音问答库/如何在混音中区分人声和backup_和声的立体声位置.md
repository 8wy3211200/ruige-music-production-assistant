# 如何在混音中区分人声和backup/和声的立体声位置

**症状：** 人声和backup（和声）全部挤在声场中间，导致层次不清晰，像两个歌手站在同一位置唱歌。

**原因：** 没有利用立体声宽度和深度来区分主次人声。主人声需要"近"和"居中"，而和声/backup需要"远"和"偏"。

**解决：** 背景人声没有固定的 dB 差值。先确认它在当前段落是绿叶、明显合唱还是呼应点缀，再调到符合这个角色的位置：通常不抢主唱注意力；需要让听众明确感到合唱时，可以更靠前。最后在音量匹配下对比确认。

**常见误区：** 把所有重要人声都放在正中间，导致声场拥挤。应该主动分配空间，就像舞台上主唱站中间、合声站两边一样。

**关键词：** #立体声, #宽度, #深度, #人声, #和声
