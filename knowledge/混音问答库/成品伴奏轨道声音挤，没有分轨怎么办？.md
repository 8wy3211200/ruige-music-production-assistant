# 成品伴奏轨道声音挤，没有分轨怎么办？

**症状：** 混音中拿到的是一条成品伴奏，没有乐器分轨，声音挤在一起难以分离。

**原因：** 没有分轨意味着无法通过声像摆位、频段调整等手段进行精细处理，可操作空间受限。

**解决：** 使用 Ozone 的 Imager 插件，将频段分为低频、中低频、中高频、高频四段，分别拉宽各频段立体声宽度。例如中频段（吉他、钢琴所在频段）适当拉宽。插件也有预置如"Enhance Space"可一键拓展空间感。

**常见误区：** 一上来就用插件处理，而不是先判断是否有其他方法。实际上插件处理立体声宽度是最暴力的手段，应是最后选项。

**关键词：** #立体声 #宽度 #插件 #贴唱
