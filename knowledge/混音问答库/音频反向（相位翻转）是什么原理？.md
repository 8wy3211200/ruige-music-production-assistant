# 音频反向（相位翻转）是什么原理？

**症状：** 用户对"反向"概念不清楚，误以为是EQ或某种滤波操作

**原因：** 声音由物体振动产生，反向是将振动方向取反。波形上表现为：原来凸起变凹陷，凹陷变凸起，即"波峰变波谷，波谷变波峰"

**解决：** 1. 在DAW中找到Utility插件（如Apple LPX的Utility）
2. 找到"反向/Phase"按钮并点亮
3. 在Cubase/Qbase中：进入前置通道设置(Pre-Fader Channel Settings)，选中后轨道会出现相位开关，直接点击即可
4. 无Utility时，任意通道条插件通常都有反向功能

**常见误区：** 误以为反向是某种EQ处理，或必须用Q3等特定插件才能操作。实际上任何带反向功能的插件都能实现

**关键词：** #插件 #动态
