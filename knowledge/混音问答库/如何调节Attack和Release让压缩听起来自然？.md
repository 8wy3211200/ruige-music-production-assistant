# 如何调节Attack和Release让压缩听起来自然？

**症状：** 压缩后声音听起来不自然，音头被压掉，声音一冒头就被按下去，一小下去又松开。

**原因：** Attack和Release设置不当，参数太快导致声音没有呼吸感。

**解决：** 1）先把Attack调到最快，让声音一触发就压缩，找到需要的压缩量；2）再把Release调到最快；3）逐步增加Attack时间，让每个字的音头露出来，通过耳朵判断；4）再调整Release，找到"把每个字都压住但不影响下一个字音头"的感觉。目标是：压缩效果达到了，但听不出明显的压缩痕迹。

**常见误区：** 不敢增加Attack时间保护音头，或者Release设置不当导致声音"喘气"感明显。

**关键词：** #压缩, #人声
