# 真鼓各轨道的处理顺序是什么？

**核心要点：** 真鼓处理顺序：kick/snare → overhead → hi-hat等补充轨道 → tom → room/ambience。

**详细解答：** 先处理kick和snare建立基本律动和音量平衡。然后处理overhead（头顶立体声麦克风），提供高频打击乐和整体空间感。接着用hi-hat等单独轨道补充细节。tom根据其在音乐中的角色（过渡或主角）相应处理。最后加入room/ambience轨道作为"粘合剂"，增强真实感和空间统一性。overhead中的串音需保留，这是真实感的一部分。各轨道比例调整比单独音色处理更重要。

**应用场景：** 真鼓分轨处理顺序、多麦克风协调、真实鼓组构建

**关键词：** #真鼓处理 #处理顺序 #轨道协调 #比例平衡
