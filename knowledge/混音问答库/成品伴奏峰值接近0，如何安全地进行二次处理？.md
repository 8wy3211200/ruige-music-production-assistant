# 成品伴奏峰值接近0，如何安全地进行二次处理？

**症状：** 成品伴奏已经是母带状态，峰值几乎到0dB，想在宿主里对伴奏做EQ、压缩等二次处理时，处理空间不足。

**原因：** 成品音乐经过母带处理后通常不会留动态空间，直接在峰值附近施加动态类效果器会导致削波。

**解决：** 使用双Mixer法处理——第一个Mixer减9dB留出动态空间，中间接入所有效果器，第二个Mixer加9dB恢复原始电平。效果器参数建议：多段压缩在副歌处预留6-12dB动态余量。若嫌麻烦也可直接用音频块的Clip Gain功能下拉音频块。

**常见误区：** 不做Gain Structure直接上动态效果器，或只加9dB不先减9dB导致处理后整体音量过大。

**关键词：** #贴唱, #动态, #混响, #导出
