# 如何在混音中系统性地提升响度？

**症状：** 混音完成后响度不足，需要提升响度但担心影响音质

**原因：** 响度提升需要分步骤、渐进式操作，避免单一步骤过度处理导致失真

**解决：** 1. **第一步：** 先使用EQ或多段压缩做精细化调整，让整体音乐更融合
2. **第二步：** 用极端斜率切掉20Hz以下低频（人耳听不到但占能量）和超高频
3. **第三步：** 使用Clipper或限制器提升响度；如果提升量较大，可分担给多个工具（如先用肥波ProR2做前期提升，再用Ozone做第二步响度追量化）
4. 提升过程中一边对比参考一边看表，根据音乐类型调整目标值

**常见误区：** 跳过前两步直接用限制器猛拉响度，容易造成音质损失和伪影

**关键词：** #响度, #EQ, #动态, #母带
