# 压缩器如何让音量不均匀的声音变得统一

```markdown
## Q1: 压缩器如何让音量不均匀的声音变得统一
**症状：** 1、2、4声音小，3声音大的鼓点或其他素材，动态差距明显
**原因：** 原始素材中不同音符/音节的电平不一致，需要通过压缩来控制动态
**解决：** 1. 设置合适的阈值（Threshold），让阈值为一条"界限线"，只压缩超过这条线的部分（如音3）；2. 小的部分（1、2、4）不被压缩；3. 用 Make-up Gain 或 Output Level 补偿被压掉的音量约2-3dB；4. 压缩器不影响阈值以下的内容，只处理超过阈值的部分
**常见误区：** 认为压缩和增益是同一个操作。实际上压缩是"炒菜"，增益是"装盘"，是流水线关系，增益不会影响压缩效果
**关键词：** #压缩, #动态

## Q2: 为什么压缩后要提升输出增益
**症状：** 压缩完成后整体音量变小，声音变"瘪"
**原因：** 压缩器把超过阈值的部分压下去了，整体响度变低，动态虽然变小了但音量也变小了
**解决：** 在压缩器输出端用 Make-up Gain 或 Output Level 提升约2-3dB。提升输出增益不影响压缩本身，只是流水线最后一步"装盘"，不会改变之前压缩的效果
**常见误区：** 混淆压缩量和输出增益的关系，或者试图通过提高增益来"弥补"压缩量。压缩解决动态问题，增益解决响度问题，职责不同
**关键词：** #压缩, #响度
```
