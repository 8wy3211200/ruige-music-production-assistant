# 如何正确设置平行压缩的Attack和Release参数？

**症状：** 用户不清楚平行压缩时Attack和Release应该怎么设置，用Pro-C时参数调整后看不出效果。

**原因：** 平行压缩的Attack决定保留多少音头动态，Release需要跟随音乐速度匹配，否则压缩器工作会不自然。

**解决：** 按速度设置Release：130BPM的歌曲，Release约400ms。Attack给一个音头即可。Ratio可设10:1左右，压缩量控制在12dB左右（此为较大值，必要时可更保守）。压缩后再通过干湿比（Dry/Wet）把未压缩信号回调，具体比例从零开始慢慢加，直到人声整齐但不突兀。

**常见误区：** #压缩, #动态, #人声
