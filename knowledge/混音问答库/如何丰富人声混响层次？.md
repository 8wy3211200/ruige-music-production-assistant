# 如何丰富人声混响层次？

**症状：** 人声混响只有基础一层，显得单薄。

**原因：** 缺少混响类型的变化和创意性用法。

**解决：** 1. **甩尾混响**：在长音或结尾处画开关，让混响时间突然变长后自然衰减
2. **Slap Delay**：用delay做左右两边按时间调整，不开feedback
3. **混响组合**：一个room/hall混响 + 一个slap + 一个甩尾，形成丰富的空间层次
4. **参考模板**：实际制作中，的混响套装组合可以多尝试

**常见误区：** 只用单一类型的混响；不敢尝试夸张的混响参数；忽视automation在混响中的运用。

**关键词：** #混响, #深度, #自动化, #宽度
