# 路由关系梳理的正确顺序是什么？

**症状：** 做完混音发现做了很多无用功，不知道先后顺序。

**原因：** 没有按照正确的工作流程操作，导致后期要返工。

**解决：** 1. **先梳理路由关系**：明确每个轨道、发送、返回的连接逻辑
2. **再做音色处理**：EQ、压缩、饱和等基础处理
3. **然后添加效果器**：混响、delay、调制等
4. **最后检查平衡**：音量平衡、左右平衡、整体融合度
5. **按照这个顺序**：路由→EQ→压缩→饱和/激励→多段压缩→混响→平衡

**常见误区：** 一上来就加各种效果器而不梳理路由；做完大量工作发现路由有问题要返工；效果链顺序混乱。

**关键词：** #插件, #人声, #融合
