# 真鼓的力度感被吉他盖住怎么办？

**症状：** 副歌部分kick和snare的力量感不足，底鼓的冲击感消失，snare的打击感弱。吉他却很大一坨顶在前面。

**原因：** 吉他占据过多中高频能量，遮盖了鼓组；同时对真鼓做了过多不必要的处理，破坏了原声乐器的自然特性。

**解决：** 1. 优先降低吉他的整体音量或用EQ削减2k-4kHz的尖锐感
2. 鼓和bass应该作为"骨架"放在前面，吉他作为"外层的皮"稍靠后
3. 对于真鼓kick：可能只需做低切（80Hz以下），先感受原始音色再逐步调整
4. 对于真鼓snare：保留击打的物理感，不要过度压缩

**常见误区：** 对真鼓进行过多效果处理（压缩、EQ等），反而破坏了真实乐器的自然动态；或者把吉他做成最突出的声音而让节奏组退后。

**关键词：** #动态, #前后关系, #低频, #压缩
