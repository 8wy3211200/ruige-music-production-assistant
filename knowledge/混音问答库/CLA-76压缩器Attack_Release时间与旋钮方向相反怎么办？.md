# CLA-76压缩器Attack/Release时间与旋钮方向相反怎么办？

**症状：** 刚接触Waves CLA-76（FET压缩器）时，不知道如何正确设置Attack和Release时间，调参方向经常搞反。

**原因：** FET压缩器的Attack/Release时间参数与旋钮数值呈反向关系——旋钮数值设置越小，实际响应越慢；旋钮数值设置越大，实际响应越快。这是模拟原始硬件电路特性的设计。

**解决：** 记住一个原则：想要响应快就把旋钮往大数值调，想要响应慢就把旋钮往小数值调。默认值通常在中间位置（4-5左右），人声处理时根据听感微调即可。操作流程：先用Input/Output平衡整体音量 → 选择Ratio（人声一般4:1）→ 调整Attack/Release时间。

**常见误区：** 误以为旋钮顺时针拧就是变快，实际上是相反的。新手容易按照普通插件的习惯去调节，导致声音要么压缩过度要么没有压缩效果。

**关键词：** #压缩, #插件
