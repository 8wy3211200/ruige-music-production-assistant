# 如何做出“大但融合”的大混响效果

**症状：** 混响听起来不融合、不自然，像在浴室唱歌一样有明显的人工感，听众会察觉到空间问题。

**原因：** 追求大混响时使用单一大型混响插件，量大且缺少层次变化。

**解决：** 采用“混响叠加法”构建丰富空间：
1. 叠加多个不同类型的混响：板式/房间混响作为基底
2. 叠加测链混响增加空间感
3. 叠加 delay 增加深度和层次
4. 叠加巨尾发送式混响补充氛围
5. 部分句子可加一点直接插入式混响调干湿比
6. **每个混响的量都要相对较小**，叠加后才显得自然且富有层次

**常见误区：** 用一个很大的混响就想做出大空间感；每个混响都想给很大的量；没有理解“大”和“明显”的区别。

**关键词：** #混响, #融合, #深度, #前后关系
