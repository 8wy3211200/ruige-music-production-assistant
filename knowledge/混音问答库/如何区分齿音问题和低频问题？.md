# 如何区分齿音问题和低频问题？

**症状：** 用户担心人声齿音重，反复提到齿音问题。

**原因：** 制作人员实际聆听后认为齿音并不明显，怀疑用户描述的"齿音重"实际是低频问题（如风声、喷麦声），可能由录音时离麦克风太近导致。

**解决：** 1. 先单独回放人声音轨，确认问题频段
2. 低频问题：用动态EQ在低频区域（如地平/低频部分）做适当衰减
3. 齿音问题：可在500-2000Hz范围用EQ处理，或使用齿音消除插件
4. 注意区分"S"类高频齿音与低频气流声

**常见误区：** 将低频喷麦/风声误判为齿音，盲目使用高频齿音消除插件，导致人声变闷或高频损失。
