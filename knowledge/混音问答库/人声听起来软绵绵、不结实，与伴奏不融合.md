# 人声听起来软绵绵、不结实，与伴奏不融合

**症状：** 人声进入后显得闷、虚、软，与伴奏中明亮刺激的旋律音色形成对比，听感上人声和伴奏是分离的，不像卡在一起的整体。

**原因：** 伴奏动态紧凑、结实，而人声本身动态起伏较大、较虚，两者音色明亮程度不匹配。音色是相对概念，当其他乐器音色很亮时，人声必须能匹配。

**解决：** 1. **大胆使用压缩**：平行压缩和多段压缩都可以放心尝试，尤其是针对人声基频和攻击性频段
2. **极端平行压缩设计**：用一轨专门做高压缩量（如表头到 -10dB 至 -20dB），发送量调小，能让人声变结实
3. **低频饱和+压缩**：对低频部分先做饱和处理，再进行压缩，增加厚实感
4. **画音量线**：适当修饰人声音量，让人声更稳定

**常见误区：** 担心压缩会让声音变"机架感"，不敢下手。实际上只要最终听感自然、人声变稳即可，可以边尝试边判断是否有机架感。

**关键词：** #人声, #动态, #压缩, #融合, #贴唱
