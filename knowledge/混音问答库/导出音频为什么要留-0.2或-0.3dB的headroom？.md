# 导出音频为什么要留-0.2或-0.3dB的headroom？

**症状：** 不确定导出时是否需要留出空间，看到有些已发布音乐没有留。

**原因：** 48kHz/24bit导出后下转为44.1kHz/16bit或320kbps MP3时，波形会发生变化。如果没有预留空间，波形凸起可能超过0dB导致失真。

**解决：** 导出时在总线输出上保留-0.2到-0.3dB的余量。这个小空白区域作为安全缓冲，无论目标格式是44.1k/16bit还是MP3，都能有效避免波形变化引发的失真问题。建议每首作品都养成这个习惯。

**常见误区：** 认为已发布歌曲没留这个空间就可以不做。实际上平台下转后波形可能已占满0dB，这是成品状态，不是标准操作。

**关键词：** #母带, #导出
