# 混响基础是什么？

**核心要点：** 混响是改变声场的空间类效果器，用于模拟不同环境中的声音反射效果。

**详细解答：** 混响（Reverb）是混音中最重要的空间类效果器之一，用于模拟声音在不同空间环境中的反射效果。它可以改变声音的深度、宽度和环境感，让声音听起来像是在特定空间（如房间、大厅、教堂等）中录制或播放的。混响有两种主要使用方式：直接插入轨道使用，或以发送方式添加效果。在人声混音中，为了不影响已经调节好的干声，通常选择发送式混响。

**关键特点：**
1. 模拟空间环境的声音反射
2. 增加声音的深度和空间感
3. 分为插入式和发送式两种用法
4. 参数包括衰减时间、预延迟、混响类型等

**应用场景：**
1. 为人声添加空间感和深度
2. 为乐器创造特定的环境氛围
3. 统一不同音源的空间感
4. 修复干涩、缺乏空间感的声音

**关键词：** #混响 #空间效果 #声场设计 #混音基础
