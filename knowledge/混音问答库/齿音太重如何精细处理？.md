# 齿音太重如何精细处理？

**症状：** 干声齿音比较重，使用去齿音工具后效果仍不够理想

**原因：** 通用去齿音插件无法精准针对每个齿音点，可能影响整体音色自然度

**解决：** 1. **EQ精准衰减**：用EQ找到齿音频点（通常5-8kHz），单独衰减约12dB，Q值适中
2. **画自动化开关**：只在有齿音的位置开启EQ衰减，画开关自动化，逐个处理
3. **音头直接衰减**：将音频块切开，直接拉上沿衰减音头（可mute问题段落）
4. **音量automation**：用Mixtool等工具单独画问题句的音量，适当降低

**常见误区：** 完全依赖去齿音插件，不敢手动精细处理；一次衰减太多导致音色发闷

**关键词：** #齿音, #EQ, #自动化, #人声
