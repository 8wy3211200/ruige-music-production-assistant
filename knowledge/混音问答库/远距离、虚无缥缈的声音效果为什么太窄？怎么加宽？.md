# 远距离、虚无缥缈的声音效果为什么太窄？怎么加宽？

**症状：** 制作远距离感的声音（如电话音、虚无缥缈的人声）时，效果集中在中间，宽度不够，缺乏空间感。

**原因：** 远距离感的混响效果通常需要更宽的声场表现，完全居中的窄声场反而违背了"远"的听觉逻辑。

**解决：** 1. 使用宽度（Width）参数更大的混响效果器，或在混响后加立体声增强插件
2. 混响的早期反射（Early Reflections）可以适当加宽
3. 可考虑添加少量双耳（Binaural）或MS处理增加空间感
4. 整体宽度建议增加到原始信号的120%-150%

**常见误区：** 认为所有效果都要做窄，远声音反而要用窄的；过度加宽导致声音相位问题。

**关键词：** #立体声 #宽度 #混响 #深度
