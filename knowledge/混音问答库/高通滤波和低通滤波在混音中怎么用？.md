# 高通滤波和低通滤波在混音中怎么用？

**核心要点：** 高通滤波用于去除不必要的低频，低通滤波用于去除不必要的高频，两者都是清理频率空间、提高混音清晰度的基础工具。

**详细解答：** 高通滤波器（High Pass Filter）允许高频通过，削减低频。在混音中的应用：1) 去除人声、吉他等非低频乐器的超低频噪音（通常设置在80-120Hz）；2) 为混音创造清晰的低频空间，让贝斯和底鼓更突出。低通滤波器（Low Pass Filter）允许低频通过，削减高频。应用包括：1) 去除高频嘶声、空气噪音；2) 创造复古或电话音效；3) 控制过于明亮的声音。使用时要设置适当的斜率（如12dB/octave或24dB/octave）和截止频率，避免过度削减影响音色完整性。

**应用场景：** 1. 清理各轨道的频率范围 2. 创造清晰的频率分层 3. 去除录音中的环境噪音
