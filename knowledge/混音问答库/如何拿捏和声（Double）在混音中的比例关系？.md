# 如何拿捏和声（Double）在混音中的比例关系？

**症状：** 副歌部分三个人的和声听起来过于明显，像合唱而非点缀，喧宾夺主。

**原因：** 没有想清楚和声的处理逻辑——是作为主要声部还是仅作为空间填充。Double和声的处理核心是"去掉不好听、加上听不出"。

**解决：** 1. 明确和声定位：仅作为氛围填充时应尽量降低存在感
2. 降低和声轨的音量至刚好能感受到"厚度"的位置
3. 可用EQ适当收窄中频，使和声更"贴"而不"跳"
4. 用声像微微偏移而非完全居中，制造空间感但不抢主唱

**常见误区：** 觉得和声轨越多越好，或者音量与主唱齐平，导致副歌变成大合唱。

**关键词：** #人声, #和声, #融合, #立体声, #前后关系
