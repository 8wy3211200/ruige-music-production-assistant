# Trilian 安装与资料库管理

## 核心概念
- Trilian 是 Spectrasonics 低音乐器平台，资料库和授权状态决定旧工程能否正常打开。
- 系统要求为 2.4 GHz 以上处理器、8 GB RAM 推荐、36 GB 磁盘空间，下载安装过程需要约 72 GB。
- Mac 需要 macOS 10.15+，Windows 需要 Windows 10+，均为 64-bit host。
- 安装可走 USB 或下载，安装后需要完成 Authorization。
- Standalone 可在不打开 DAW 时检查键盘、节拍器和库路径。

## 使用场景
- 第一次部署贝斯音源、换电脑、移动 STEAM 目录、检查旧工程缺失音色。
- 演出机或移动编曲机需要确认外置盘速度和授权。
- 项目交付需要记录 Trilian 版本、库路径和使用 patch。
- 大工程加载慢时，先检查资料库位置而不是马上删插件。

## 核心参数
- Processor：2.4 GHz+。
- RAM：8 GB 或更多推荐。
- Disk：36 GB，可下载安装时约 72 GB。
- Formats：AU、VST3、VST2、AAX，按宿主统一。
- STEAM Directory：移动库后重新定位。
- Smart Update：旧工程出现浏览器或 patch 异常时先更新。

## 实战案例
- 新电脑开旧工程前，先在 Standalone 加载几个常用 acoustic/synth bass patch。
- 外置盘尽量用 SSD 或 USB 3，避免贝斯 round robin 和长音加载卡顿。
- 协作交付时把贝斯音色另存用户 patch，并记录是否用了 Multi。
- 演出模板只加载需要音色，减少无关库扫描。
