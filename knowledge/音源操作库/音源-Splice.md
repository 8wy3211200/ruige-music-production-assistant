# Splice 操作指南

**音源简介**：Splice是创意采样平台，提供海量采样和循环，支持搜索、试听、下载

**音色组结构**：
1. **鼓采样**：底鼓、军鼓、踩镲、通鼓、打击乐
2. **乐器采样**：贝斯、吉他、键盘、弦乐、管乐
3. **人声采样**：人声片段、和声、效果人声
4. **合成器采样**：合成器音色、效果音色
5. **循环片段**：鼓循环、乐器循环、完整乐句
6. **效果采样**：上升、下降、冲击、过渡效果

**常用技法对应的音色选择**：
- **采样鼓使用**：搜索风格匹配的鼓采样
- **音色补充**：查找缺少的乐器音色
- **创意激发**：浏览随机采样获取灵感
- **快速构建**：使用循环片段快速搭建编曲
- **效果增强**：添加效果采样增强过渡

**操作流程**：
1. **搜索**：使用关键词、风格、BPM、调性过滤
2. **试听**：在线试听，标记喜欢和下载
3. **下载**：下载到本地，整理到采样库
4. **导入**：导入DAW或采样器使用
5. **处理**：时间拉伸、音高调整、效果处理

**实战案例**：制作Trap beat：在Splice搜索"Trap Kick 808"下载底鼓采样，搜索"Trap Snare"下载军鼓采样，搜索"Trap HiHat"下载踩镲采样，在DAW中组合编写
