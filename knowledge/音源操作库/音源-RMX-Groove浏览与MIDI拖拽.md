# RMX Groove 浏览与 MIDI 拖拽

## 核心概念
- Stylus RMX 的核心不是单个鼓采样，而是 groove、slice 和 MIDI 驱动。
- 浏览器用 Suite、Element、Groove 等层级组织素材，适合快速找律动。
- MIDI 拖拽能把 groove 转进 DAW，后续可改音符和编曲结构。
- 先找接近的律动，再在 DAW 里删减，而不是从零编每个切片。
- Groove 选择要和歌曲速度、重拍位置、鼓组密度一致。

## 使用场景
- 电子律动草稿、影视节奏底、percussion loop 拆分、鼓组辅助层。
- 需要从 loop 变成可编辑 MIDI 时，用拖拽方式进 DAW。
- 需要保留 RMX 音色但改节奏时，只改 MIDI。
- 需要替换声音不换律动时，保持 MIDI 片段，更换 groove 或 element。

## 核心参数
- 需要避免版权式完整 loop 感时，重排切片和删减密度。
- Browser：按库、风格、元素浏览 groove。
- MIDI Drag and Drop：把 groove 作为 MIDI 片段放入工程。
- Edit Groups：控制不同切片组的处理。
- Part：多 groove 组合时的独立槽位。
- Tempo Sync：让 groove 跟随工程速度。

## 实战案例
- Preview：先试听密度和重拍，再决定是否拖入。
- 流行副歌：找接近 shaker/percussion groove，拖入后删掉抢人声的切片。
- 影视紧张段：选择低频少的 texture groove，配合大鼓单独写重拍。
- 电子律动：保留 RMX MIDI，换不同 element 比较音色。

## 补充提醒
- 避免机械：在 DAW 里微调少数切片 timing，而不是全量量化。
