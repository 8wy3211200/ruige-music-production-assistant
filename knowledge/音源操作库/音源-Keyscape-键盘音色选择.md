# Keyscape 键盘音色选择

## 核心概念
- Keyscape 的强项是键盘族群，不是只找一个钢琴预设。
- 浏览器按 patch、模型和属性筛选，能快速从大库里缩小选择。
- 声学钢琴、电钢、Clav、Celeste、Toy Piano 等各自承担不同编曲角色。
- 音色选择先看歌曲功能：主体、伴奏、复古颜色、点缀或低音。
- Auto-Play 可快速试听，但最终要用自己的 MIDI 力度验证。

## 使用场景
- 流行钢琴主体需要 LA Custom C7 或类似全频钢琴。
- 复古 R&B 需要 Rhodes、Wurlitzer、Clavinet 这类有中频性格的键盘。
- 童话、Lo-fi、氛围段落可从 Toy Piano、Celeste、Chimeatron 找颜色。
- 低频 hook 可从 Rhodes Piano Bass 或 Tine Bass 补一个键盘低音层。

## 核心参数
- Browser Filters：用类型、模型、风格标签缩小范围。
- Search Field：直接搜 Rhodes、Wurli、C7、Toy 等目标关键词。
- Auto-Play：快速初筛，避免每个 patch 都手动弹一遍。
- Info：查看音色模型和控制说明，别只看预设名字。
- Favorites/Rating：把项目常用音色标记出来，形成个人可复用库。
- Patch Save：改过控制和效果后另存，避免覆盖原厂判断。

## 实战案例
- 抒情主歌：C7 降低亮度，留中低频身体，不要把钢琴做得太宽。
- 副歌复古层：Rhodes 低切后混在钢琴下方，提供温暖和晃动。
- 桥段点缀：Toy Piano 用短音和延迟，只占高频缝隙。
- 贝斯补层：Piano Bass 只保留 80-200 Hz 的音头，和电贝斯错开。
