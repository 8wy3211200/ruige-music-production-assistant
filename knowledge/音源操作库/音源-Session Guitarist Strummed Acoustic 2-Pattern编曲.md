# Session Guitarist Strummed Acoustic 2 Pattern 编曲

## 核心概念
- Pattern 是 Strummed Acoustic 2 的编曲核心，决定吉他在歌曲里的节奏身份。
- 同一和弦进行可以通过 pattern 密度、重音和段落切换产生完整编曲。
- 不要全曲使用同一个 pattern，否则真实但无发展。
- Pattern 选择要和鼓、贝斯、主唱咬合，不是越复杂越好。
- 先确定主歌、副歌、过门三种密度，再细调音色。

## 使用场景
- 从 demo 快速成型到正式编曲都适用。
- 主歌需要轻、少、靠后；副歌需要亮、宽、强。
- 过门需要短暂停顿或上行型节奏提示段落。
- 双吉他时两轨 pattern 要互补，不要完全同步。

## 核心参数
- 需要自然收尾时，用 ending 或单次 strum。
- Pattern Slot：存放可切换节奏型。
- Intensity：控制扫弦强弱和密度。
- Swing/Feel：匹配歌曲律动。
- Accent：让重拍和弱拍形成真实手感。
- Humanize：保留微小偏移，避免 MIDI 块感。

## 实战案例
- Sound：用 pick、body、stereo 或效果决定位置。
- 民谣主歌：低强度 pattern，加少量 swing，和贝斯错开重音。
- 流行副歌：双轨左右，各自不同 pattern，一个重扫一个补切分。
- Bridge：去掉连续扫弦，只保留半小节回答。

## 补充提醒
- 最后副歌：pattern 不变但亮度和电平略推，让段落更大。
